**免疫球蛋白 A（Immunoglobulin A, IgA）** 是体液免疫系统的核心成员，约占人体免疫球蛋白总量的 **10-15%**，但它是**粘膜免疫**的主力军。以下从基础定义、异常解读到临床应对为您系统梳理：

---

### 一、指标基本涵义

**1. 生理功能**
- **位置特殊**：分为**血清型**（血液）和**分泌型（sIgA）**（唾液、泪液、乳汁、呼吸道/消化道/泌尿生殖道粘膜表面）。
- **第一道防线**：sIgA 像"粘膜胶水"，阻止细菌、病毒、过敏原粘附在上皮细胞，是呼吸道和肠道免疫的"守门员"。
- **免疫调节**：参与维持肠道菌群平衡（防止炎症性肠病）。

**2. 参考范围**（因实验室方法不同而异）
- **成人血清 IgA**：通常 **0.7–4.0 g/L**（或 70–400 mg/dL）
- **年龄差异**：新生儿几乎为零（通过胎盘转移极少），6 个月至青春期逐渐达成人水平，老年人可能轻度下降。

---

### 二、常见异常及可能原因

#### **情况 A：IgA 升高（高 IgA 血症）**
| 常见原因 | 详细说明 |
|---------|---------|
| **肝脏疾病** | **酒精性肝病**特征性表现（酒精刺激肠道产生 IgA）；肝硬化、慢性肝炎 |
| **慢性感染** | 结核、慢性支气管炎、HIV 慢性期、寄生虫感染（肠道/肝脏） |
| **自身免疫病** | 类风湿关节炎、系统性红斑狼疮（SLE）、干燥综合征（Sjögren）、IgA 血管炎（Henoch-Schönlein 紫癜） |
| **IgA 肾病** | 亚洲人群最常见肾小球疾病（Berger 病），表现为血尿/蛋白尿 |
| **血液系统肿瘤** | IgA 型多发性骨髓瘤、淋巴增殖性疾病 |
| **炎症性肠病** | 克罗恩病、溃疡性结肠炎活动期 |

#### **情况 B：IgA 降低（低 IgA 血症）**
| 常见原因 | 详细说明 |
|---------|---------|
| **选择性 IgA 缺乏症** | **最常见的原发性免疫缺陷**（人群患病率 1:500–1:700），多数无症状，部分反复感染 |
| **药物/医源性** | 糖皮质激素长期使用、免疫抑制剂（环磷酰胺等）、化疗、放疗 |
| **蛋白质丢失** | 肾病综合征（尿中丢失）、蛋白丢失性肠病（肠道淋巴管扩张） |
| **获得性免疫缺陷** | 某些病毒感染（如 EV71 重症）、营养不良 |
| **其他免疫缺陷** | 常见变异型免疫缺陷病（CVID）、共济失调毛细血管扩张症 |

### 三、异常后的建议处理方案

#### **若 IgA 升高：**

**立即行动：**
1. **复查确认**：排除实验室误差（标本溶血、脂血干扰）。
2. **回顾病史**：近期感染史、饮酒习惯（尤其烈酒）、用药史（苯妥英钠等可诱发）。

**针对性检查：**
- **肝脏**：肝功能（GGT、AST/ALT）、肝脏超声（排查酒精性/脂肪性肝病）
- **肾脏**：尿常规（看红细胞形态）、24 小时尿蛋白、肾功能（排查 IgA 肾病）
- **免疫筛查**：抗核抗体谱（ANA）、类风湿因子（RF）、免疫固定电泳（排查单克隆升高→骨髓瘤）
- **感染排查**：结核 T-SPOT、肝炎病毒载量

**生活方式调整：**
- **严格限酒**：酒精性肝病相关升高需绝对戒酒 3–6 个月后复查。
- **抗炎饮食**：减少高糖、高脂饮食（减轻肝脏和肠道炎症）。
- **控制体重**：肥胖相关脂肪性肝炎常伴 IgA 轻度升高。

#### **若 IgA 降低（<0.7 g/L）：**

**关键鉴别：**
1. **确认持续性**：需间隔 3–6 个月两次检测均低，方可诊断"选择性 IgA 缺乏"。
2. **评估感染负担**：记录过去一年呼吸道/消化道/泌尿道感染频率。

**必须检查：**
- **其他免疫球蛋白**：IgG、IgM 水平（若 IgG 也低→需考虑 CVID）
- **疫苗接种反应**：检测破伤风/白喉抗体滴度（评估 IgG 功能是否代偿）
- **自身抗体**：抗 IgA 抗体（部分缺乏者会产生，与输血过敏相关）
- **淋巴细胞亚群**：CD19/20（B 细胞数量）

**生活方式与医疗建议：**
- **感染预防**：勤洗手、避免人群密集场所、接种灭活疫苗（**避免活疫苗**如口服脊髓灰质炎，除非免疫功能评估通过）。
- **输血安全**：若需输血，必须告知医生"IgA 缺乏"，使用**洗涤红细胞**或来自 IgA 缺乏供者的血液（防止过敏性休克）。
- **营养支持**：补充锌、维生素 A（对粘膜免疫修复关键）。
- **谨慎使用**：避免不必要的免疫抑制药物（如长期外用强效激素可能系统吸收）。

---

### 四、特别警示（重点人群）

1. **IgA 肾病高发信号**：年轻男性（16–35 岁）体检发现 IgA 升高 + 尿潜血阳性，需尽快肾内科就诊，必要时肾活检。
2. **无症状 IgA 缺乏**：多数选择性缺乏者无需治疗，但需终身携带医疗警示卡（注明输血注意事项）。
3. **假阴性/假阳性**：IgA 型类风湿因子阳性（自身抗体）可能干扰检测，导致假性升高。

**重要提醒**：IgA 单项异常通常不构成独立诊断，需结合 **IgG、IgM、补体 C3/C4** 及临床症状综合判断。建议携带报告至**免疫科或肾内科**（若伴尿检异常）进一步评估。
