**Basophils, Absolute（嗜碱性粒细胞绝对值，简称 BASO#）** 是血常规（CBC）中的一项指标，指每单位体积血液中嗜碱性粒细胞的具体数量，通常以 **10⁹/L（或 cells/µL）** 表示。

以下是该指标的详细解读：

---

### 一、指标涵义
- **细胞特性**：嗜碱性粒细胞是白细胞中最少见的一种（通常占白细胞总数的 0.5%-1%），胞质内含有组胺、肝素等颗粒，参与过敏反应和抗凝血过程。
- **绝对值 vs 百分比**：绝对值（#）比百分比（%）更能真实反映嗜碱性粒细胞的实际数量，因为它不受其他白细胞（如中性粒细胞或淋巴细胞）数量变化的影响。
- **参考范围**：通常为 **0–0.1×10⁹/L**（或 0–100 cells/µL），不同实验室标准可能略有差异。

---

### 二、常见异常类型及可能原因

#### 1. **嗜碱性粒细胞增多（Basophilia）—— 绝对值 > 0.1–0.15×10⁹/L**
这是临床更关注的异常，尤其是**持续性升高**。

| 可能原因 | 具体说明 |
|---------|---------|
| **骨髓增殖性肿瘤（MPN）** | **最需要警惕**，特别是**慢性粒细胞白血病（CML）**、真性红细胞增多症（PV）、原发性血小板增多症（ET）。CML 患者常伴有显著嗜碱性粒细胞增多。 |
| **过敏性疾病** | 慢性荨麻疹、过敏性鼻炎、哮喘、药物过敏（嗜碱性粒细胞参与 I 型超敏反应）。 |
| **炎症/自身免疫病** | 溃疡性结肠炎、克罗恩病、类风湿关节炎、红皮病。 |
| **内分泌疾病** | 甲状腺功能减退（甲减）、雌激素治疗。 |
| **感染** | 某些病毒感染（如水痘、流感）、结核。 |
| **缺铁性贫血恢复期** | 补铁治疗后可能出现一过性升高。 |
| **脾切除术后** | 脾脏是破坏衰老嗜碱性粒细胞的场所，切除后可升高。 |

#### 2. **嗜碱性粒细胞减少（Basopenia）—— 绝对值接近 0**
- **临床意义较小**，通常不单独作为诊断依据。
- 可能见于：
  - 急性严重过敏反应（细胞脱颗粒后暂时耗竭）
  - 甲状腺功能亢进（甲亢）
  - 库欣综合征（皮质醇增多）
  - 急性感染或应激状态

---

### 三、异常对应的后续处理建议

#### 🔴 **如果绝对值明显升高（> 0.2–0.3×10⁹/L 或持续上升）**
1. **血液科就诊**：排除骨髓增殖性肿瘤，特别是如果伴有：
   - 白细胞总数（WBC）升高
   - 血小板增多或贫血
   - 脾脏肿大（腹部超声）
   - 盗汗、体重下降、腹部饱胀感
2. **进一步检查**：
   - **外周血涂片**：观察细胞形态
   - **BCR-ABL 融合基因检测**：排除 CML
   - **JAK2/CALR/MPL 基因突变检测**：排除其他 MPN
   - **骨髓穿刺活检**：如高度怀疑血液系统疾病
   - **腹部超声**：检查脾脏大小

#### 🟡 **如果轻度升高（0.1–0.2×10⁹/L）且无其他症状**
1. **1–2 周后复查**：排除实验室误差或一过性因素（如过敏发作期、感染恢复期）。
2. **观察伴随症状**：是否有皮疹、鼻痒、腹泻、乏力等。
3. **回顾用药史**：是否服用雌激素类药物或抗甲状腺药物。

#### 🟢 **生活方式调整（适用于过敏相关升高）**
- **避免过敏原**：尘螨、花粉、宠物皮屑、特定食物。
- **环境控制**：使用空气净化器，保持室内清洁，避免潮湿（减少霉菌）。
- **饮食**：减少高组胺食物（如发酵食品、加工肉类）的摄入，如有明确食物过敏需严格忌口。
- **记录症状日记**：记录嗜碱性粒细胞升高时伴随的身体反应，帮助医生判断病因。

#### ⚪ **如果减少**
- 通常无需特殊处理，建议复查即可。如有甲亢或长期使用激素病史，应针对原发病管理。

---

### 四、特别提醒
- **不要孤立看待**：需结合白细胞总数（WBC）、嗜酸性粒细胞（Eosinophils）、血小板（PLT）、血红蛋白（Hb）等指标综合判断。
- **假性升高**：极少数情况下，其他细胞碎片或实验室技术问题可能导致假性嗜碱性粒细胞增多，复查可验证。
- **儿童与成人差异**：儿童嗜碱性粒细胞绝对值通常较成人更低，且对过敏的反应可能更敏感。

**总结**：单纯的轻度嗜碱性粒细胞绝对值升高（尤其 < 0.2×10⁹/L）多为良性或过敏相关，定期复查即可；**但若显著升高或伴随其他血细胞异常，必须及时排查骨髓增殖性疾病。**
