## 二、常见关键指标示例

### 1. **血压 BP（mmHg）**
- **定义**：血液对血管壁的侧压力，通常记录收缩压/舒张压
- **正常值**：<120/80 mmHg（理想），<140/90 mmHg（正常高值/高血压前期需关注）
- **异常**：
  - **偏高（高血压）**：≥140/90 mmHg
    - *原因*：遗传、高盐饮食、肥胖、饮酒、肾病、内分泌疾病
    - *建议*：限盐（<5g/天）、减重、规律有氧（快走/游泳）、戒烟酒；≥160/100或伴头痛/心悸需心内科就诊
  - **偏低**：<90/60 mmHg
    - *原因*：体质性低血压、脱水、出血、甲减
    - *建议*：若无头晕乏力可观察；有症状需排查心脏和内分泌问题

### 2. **空腹血糖 FPG（mmol/L）**
- **定义**：空腹8-10小时后静脉血浆葡萄糖浓度
- **正常值**：3.9-6.0 mmol/L；6.1-6.9为空腹血糖受损（糖尿病前期）；≥7.0考虑糖尿病
- **异常**：
  - **偏高**：
    - *原因*：胰岛素抵抗、糖尿病、应激状态、前晚暴饮暴食、药物（激素）
    - *建议*：复查空腹+餐后2小时血糖+糖化血红蛋白（HbA1c）；控制精制碳水、减重、每周150分钟中等强度运动
  - **偏低**（<3.9）：
    - *原因*：降糖药过量、胰岛素瘤、长期饥饿、肝衰竭
    - *建议*：立即进食糖块；反复发作需查胰岛素水平和腹部CT

### 3. **低密度脂蛋白胆固醇 LDL-C（mmol/L）**
- **定义**："坏胆固醇"，动脉粥样硬化主要危险因素
- **目标值**：健康人群<3.4；高危人群（高血压/糖尿病）<2.6；冠心病患者<1.8
- **异常**：
  - **偏高**：
    - *原因*：高脂饮食、甲减、肾病综合征、家族性高胆固醇血症
    - *建议*：减少饱和脂肪（动物内脏、肥肉）、增加膳食纤维（燕麦、豆类）；若>4.9或伴心血管病史需他汀类药物治疗
  - **偏低**（<1.0）：
    - *原因*：营养不良、甲亢、慢性消耗性疾病
    - *建议*：排查甲亢；适当增加健康脂肪（坚果、深海鱼）

### 4. **谷丙转氨酶 ALT（U/L）**
- **定义**：肝细胞内酶，肝细胞损伤时漏出至血液
- **正常值**：男性<50，女性<40（各实验室略有差异）
- **异常**：
  - **偏高**：
    - *轻度*（<3倍上限）：脂肪肝、饮酒、熬夜、剧烈运动、药物（如对乙酰氨基酚）
    - *中重度*（>3倍）：病毒性肝炎、自身免疫性肝病、胆道梗阻、心肌梗死
    - *建议*：戒酒、避免熬夜、停用可疑药物；持续升高需查乙肝五项、肝脏超声、自身免疫抗体；>80 U/L建议消化内科/肝病科就诊
  - **偏低**：通常无临床意义

### 5. **血肌酐 Cr（μmol/L）**
- **定义**：肌肉代谢产物，经肾脏排泄，反映肾小球滤过功能
- **正常值**：男性53-106，女性44-97（随年龄肌肉量变化）
- **异常**：
  - **偏高**：
    - *原因*：急慢性肾功能不全、脱水、心衰、剧烈运动、肉食过多
    - *建议*：复查并计算eGFR（估算肾小球滤过率）；查尿常规、肾脏超声；控制蛋白质摄入（0.6-0.8g/kg/天）、避免肾毒性药物（如不明成分中草药、NSAIDs止痛药）
  - **偏低**：
    - *原因*：肌肉萎缩、长期素食、妊娠
    - *建议*：增加优质蛋白摄入，排查肌肉疾病

## 三、通用就医建议

出现以下情况建议**立即或近期专科就诊**：
1. **危急值**：血糖<2.8或>22.2 mmol/L、血压>180/120 mmHg伴症状、血红蛋白<60 g/L
2. **进行性升高**：某指标连续多次体检持续上升
3. **多系统异常**：如同时存在蛋白尿+血肌酐高+血压高（提示肾病综合征）
4. **肿瘤标志物显著升高**：如CEA、AFP超过正常值5-10倍
