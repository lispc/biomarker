**可利用睾酮（Bioavailable Testosterone）** 是评估雄激素状态比"总睾酮"更精准的指标，尤其适用于中老年男性或患有影响性激素结合球蛋白（SHBG）疾病的人群。以下是详细解读：

---

### 一、指标涵义与生理意义

**定义**  
可利用睾酮 = **游离睾酮（Free T，约2%）** + **白蛋白结合睾酮（Albumin-bound T，约40%）**  
- 游离睾酮：完全游离，生物活性最强  
- 白蛋白结合睾酮：结合松散，可迅速解离发挥生理作用  
- **区别于总睾酮（Total T）**：总睾酮还包含与SHBG紧密结合的部分（约60%），这部分难以被组织利用

**为何重要**  
- **SHBG水平干扰**：老年、肝病、甲亢、药物（如抗癫痫药）会升高SHBG，导致"总睾酮正常但可利用睾酮不足"（功能性性腺功能减退）  
- **肥胖影响**：肥胖者SHBG通常降低，总睾酮可能偏低，但可利用睾酮可能相对正常或下降不明显

**参考范围**（各实验室方法不同，仅作参考）  
- **成年男性**：通常 **2.1–10.8 ng/dL**（或 **73–374 pmol/L**），随年龄增长而下降  
- **绝经前女性**：约 **0.5–2.5 ng/dL**（约为男性的1/10–1/20）

---

### 二、常见异常及可能原因

#### 1. 可利用睾酮**低下**（临床最常见）

**症状提示（男性）**  
性欲减退、勃起功能障碍、疲劳、情绪低落/抑郁、肌肉量/力量下降、内脏脂肪堆积、骨密度降低、贫血

**主要原因分类**  
| 类型 | 具体原因 |
|------|---------|
| **原发性**（睾丸本身问题） | 年龄相关衰退（男性更年期）、睾丸外伤/炎症（腮腺炎后遗症）、克氏综合征（Klinefelter）、化疗/放疗损伤 |
| **继发性**（下丘脑-垂体轴） | 垂体瘤（泌乳素瘤）、颅脑外伤/手术、血色病、过度运动/厌食症（低能量可利用） |
| **功能性/生活方式** | **肥胖（内脏脂肪促进芳香化酶将睾酮转为雌激素）**、睡眠呼吸暂停、长期压力（皮质醇升高）、酗酒、营养不良 |
| **药物影响** | 阿片类镇痛药、糖皮质激素（泼尼松等）、抗雄激素药（非那雄胺等）、部分抗抑郁药（SSRIs） |
| **系统性疾病** | 未控制的糖尿病、慢性肾病、肝硬化、甲状腺功能异常（甲亢/甲减） |

**女性低下**  
相对少见，可见于卵巢早衰、垂体功能减退、长期口服避孕药、肾上腺功能不全。

---

#### 2. 可利用睾酮**升高**

**男性升高**  
- 外源性补充（合成类固醇滥用）  
- 肾上腺或睾丸间质细胞肿瘤（罕见）  
- 先天性肾上腺增生（CAH，21-羟化酶缺乏）

**女性升高**（临床意义更大）  
- **多囊卵巢综合征（PCOS）**（最常见，伴痤疮、多毛、月经稀发）  
- 肾上腺皮质增生或肿瘤  
- 分泌雄激素的卵巢肿瘤（如Sertoli-Leydig细胞瘤）  
- 胰岛素抵抗/严重肥胖

---

### 三、发现异常后的处理建议

#### **第一步：复查确认**
- **检测时间**：睾酮分泌有昼夜节律，**早晨7-10点抽血**最准确；需空腹  
- **重复检测**：单次异常需在**不同日重复2-3次**确认，避免急性疾病、睡眠不足或酒精干扰  
- **同步检测SHBG和总睾酮**：计算游离睾酮指数（FTI）或直接测定游离睾酮，以验证结果

#### **第二步：生活方式干预（尤其轻度下降）**
- **减重**：减重5-10%可显著提升睾酮（减少脂肪组织芳香化作用）  
- **抗阻训练**：大肌群力量训练（深蹲、硬拉）刺激睾酮急性分泌和受体敏感性  
- **睡眠优化**：保证7-8小时深度睡眠，治疗睡眠呼吸暂停（CPAP治疗可恢复睾酮）  
- **压力管理**：冥想、正念减压，降低皮质醇  
- **营养调整**：补充锌（牡蛎、红肉）、维生素D（目标血清>30 ng/mL）、健康脂肪（单不饱和脂肪酸），避免长期极低脂饮食  
- **戒除酒精**：酒精直接抑制睾丸 Leydig 细胞功能

#### **第三步：医学评估（进一步检查）**
根据初步结果选择：

| 检查项目 | 目的 |
|---------|------|
| **LH + FSH** | 鉴别原发（升高） vs 继发（降低/正常）性性腺功能减退 |
| **泌乳素（Prolactin）** | 排除高泌乳素血症/垂体瘤（尤其伴头痛、视力改变） |
| **雌二醇（Estradiol）** | 评估芳香化酶活性（肥胖者常升高） |
| **甲状腺功能（TSH, FT4）** | 排除甲状腺疾病干扰 |
| **铁蛋白/转铁蛋白饱和度** | 排除血色病（铁过载损伤垂体/睾丸） |
| **空腹血糖 + HbA1c + 血脂** | 评估代谢综合征关联 |
| **睾丸超声** | 如触及肿块或原发性减退，排除肿瘤或精索静脉曲张 |
| **垂体MRI** | 仅当继发性减退且泌乳素升高，或怀疑垂体病变时 |

#### **第四步：治疗决策**

**男性睾酮替代治疗（TRT）的考量**  
- **适应症**：反复确认低下 + 明确症状 + 无禁忌证（前列腺癌/乳腺癌史、严重红细胞增多症、未控制的严重心衰）  
- **慎用/禁忌**：有生育需求者（TRT抑制精子生成，需选择HCG或克罗米芬等保留生育力的方案）  
- **监测**：治疗期间每3-6个月复查红细胞压积（HCT）、前列腺特异性抗原（PSA）、血脂、乳腺检查

**针对女性的治疗**  
- 女性高雄激素血症需排查PCOS或肿瘤，通常不使用睾酮补充（仅少数性腺功能低下女性经严格评估后局部使用低剂量凝胶）

---

### 四、特别注意事项

1. **年龄解读**：80岁男性的"正常"值与30岁不同，需结合年龄调整预期，避免过度治疗  
2. **实验室差异**：不同检测方法（平衡透析法 vs 计算法）结果差异大，建议在同一实验室随访  
3. **症状优先**：轻度下降但无症状者通常观察即可；数值"正常低限"但症状典型者，需排查其他原因（抑郁、甲状腺功能异常）  
4. **勿自行补充**：外源性睾酮会反馈抑制垂体，导致睾丸萎缩和不育，**必须在内分泌科或男科医生指导下使用**
