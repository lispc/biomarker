**卵泡刺激素（FSH）** 是由脑垂体前叶分泌的一种糖蛋白激素，是评估生殖内分泌功能和性腺（卵巢/睾丸）储备的核心指标。它与黄体生成素（LH）协同调节生殖细胞发育和性激素合成。

---

### 一、基本生理意义与正常参考值

**核心功能：**
- **女性**：刺激卵泡生长发育，促进雌激素合成；与LH协同触发排卵
- **男性**：促进睾丸支持细胞增殖，启动精子发生（生精过程）

**正常参考范围**（不同实验室略有差异，单位：IU/L）：

| 人群/状态 | 参考范围 |
|---------|---------|
| **女性（育龄期）** | |
| - 卵泡期 | 3.5–12.5 |
| - 排卵期 | 4.7–21.5 |
| - 黄体期 | 1.7–7.7 |
| **女性（绝经期）** | 25.8–134.8 |
| **男性** | 1.5–12.4 |
| **儿童** | < 3.0–5.0 |

*注：解读时必须结合月经周期阶段（女性）、年龄及其他激素（LH、雌二醇E2、睾酮T、抗缪勒管激素AMH等）。*

---

### 二、FSH 升高的异常及处理

**临床意义**：通常提示**原发性性腺功能减退**（性腺本身病变，垂体试图通过分泌更多FSH来代偿）。

#### 常见原因
**女性：**
- **卵巢储备功能下降（DOR）**：35岁以上女性常见，提示卵泡数量减少
- **卵巢早衰（POI）**：40岁前绝经，FSH常>25–40 IU/L（需两次检测间隔4周确认）
- **自然绝经期**：卵巢功能衰竭的正常生理过程
- **性腺发育异常**：如Turner综合征（45,X）
- **医源性损伤**：卵巢手术、放疗、化疗后

**男性：**
- **原发性睾丸功能减退**：克氏综合征（Klinefelter，47,XXY）、隐睾、睾丸炎（如腮腺炎性）、外伤/扭转
- **无精症/严重少精症**：生精上皮损伤导致负反馈减弱

#### 后续建议
1. **详细激素评估**：
   - 必查：LH、雌二醇（女）/睾酮（男）、AMH（女）、抑制素B
   - 绝经前女性若FSH高但E2也高，可能处于卵泡早期正常波动；若FSH高而E2低，提示卵巢功能减退
   
2. **影像学检查**：
   - 女性：阴道超声（窦卵泡计数AFC、卵巢体积）
   - 男性：睾丸超声（体积、血流、微石症）
   
3. **遗传学检查**：染色体核型分析（如怀疑Turner、Klinefelter综合征）

4. **生活方式干预**（辅助性，对器质性病变逆转有限）：
   - 戒烟（吸烟加速卵泡耗竭）
   - 补充抗氧化剂（辅酶Q10、维生素D、DHEA需医生指导）
   - 避免过度节食，保持BMI 18.5–24

### 三、FSH 降低的异常及处理

**临床意义**：通常提示**继发性（中枢性）性腺功能减退**（下丘脑-垂体轴受抑制或病变）。

#### 常见原因
**中枢性因素：**
- **下丘脑-垂体病变**：垂体瘤（泌乳素瘤最常见）、席汉综合征、空蝶鞍综合征、颅脑外伤/感染
- **功能性低促性腺激素**：
  - 过度运动（女运动员三联征）
  - 神经性厌食/营养不良（BMI<18）
  - 肥胖（BMI>30，脂肪细胞分泌瘦素干扰GnRH脉冲）
  - 慢性压力、睡眠剥夺

**药物/激素影响：**
- 口服避孕药、雌激素替代治疗（负反馈抑制）
- GnRH激动剂（如治疗子宫内膜异位症的药物）
- 雄激素过量（女性多囊卵巢综合征PCOS中，LH可能升高但FSH相对低，LH/FSH>2–3）

#### 后续建议
1. **垂体-下丘脑评估**：
   - 必查：LH、泌乳素（PRL）、甲状腺功能（TSH）、皮质醇、IGF-1
   - 影像学：垂体MRI（增强）——若PRL升高或伴有头痛、视力视野缺损
   
2. **排除生活方式因素**：
   - 调整运动强度（每周中等强度运动<5小时，避免过度有氧）
   - 体重管理：过瘦者需增重，肥胖者减重5–10%常可恢复排卵
   - 压力管理、保证睡眠

3. **药物调整**：在医生指导下评估是否需停用抑制性激素类药物

### 四、关键比值与综合解读

| 情况 | FSH | LH | 典型场景 |
|------|-----|-----|---------|
| **绝经后** | ↑↑ | ↑↑ | 卵巢功能衰竭，两者均显著升高 |
| **卵巢早衰** | ↑ | ↑或正常 | FSH/LH>2–3.6提示储备下降 |
| **PCOS（多囊）** | 正常或↓ | ↑ | LH/FSH>2–3，伴睾酮升高 |
| **垂体性闭经** | ↓↓ | ↓↓ | 两者均低，需排查垂体瘤 |

---

### 五、通用就医建议

1. **复查确认**：单次FSH异常可能因应激、急性疾病或月经周期误差导致，建议2–4周后复查（女性需在月经周期第2–4天抽血）。
   
2. **不要孤立解读**：必须结合雌二醇（E2）判断：
   - **高FSH + 低E2**：提示真性卵巢功能减退
   - **高FSH + 高E2**：可能处于围绝经期波动或检测误差

3. **紧急就医指征**：
   - 伴严重头痛、视力改变（警惕垂体瘤）
   - 年轻女性（<40岁）FSH>25 IU/L伴闭经（卵巢早衰）
   - 男性FSH显著升高伴睾丸萎缩（需排除肿瘤）

4. **生育规划**：若FSH>15 IU/L（女）或>10 IU/L（男），建议尽早咨询生殖医学中心，自然受孕几率可能下降。

*注：具体治疗方案（如激素替代、促排卵）必须由内分泌科或生殖科医生根据完整评估制定。*
