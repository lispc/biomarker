**Free Androgen Index (FAI)**，即**游离雄激素指数**，是评估体内具有生物活性的雄激素（主要是睾酮）水平的重要间接指标。由于血液中大部分睾酮与性激素结合球蛋白（SHBG）或白蛋白结合而失去活性，只有约1-3%以游离形式存在，而直接检测游离睾酮技术要求高、成本高，因此临床上常用FAI作为经济有效的替代估算指标。

---

## 一、指标涵义与计算

**计算公式：**
$$FAI = \frac{总睾酮(nmol/L)}{SHBG(nmol/L)} \times 100$$

**核心意义：**
- 反映**生物可利用睾酮**（游离睾酮+白蛋白结合睾酮）的相对水平
- 排除了SHBG浓度变化对总睾酮检测结果的干扰
- **数值越高**，代表具有生物活性的雄激素比例越高

**参考范围**（不同实验室略有差异）：
- **成年男性**：30–150
- **育龄女性**：< 5（或< 7，依实验室标准）
- **绝经后女性**：< 3

---

## 二、常见异常及可能原因

### ▶ FAI 升高（高雄激素血症）

**女性常见原因（临床更关注）：**
1. **多囊卵巢综合征（PCOS）** - 最常见，占70-80%
2. **先天性肾上腺皮质增生（CAH）** - 21-羟化酶缺乏
3. **分泌雄激素的肿瘤**（卵巢或肾上腺）
4. **高泌乳素血症** - 抑制卵巢功能，相对提高雄激素比例
5. **药物因素**：达那唑、某些孕激素、类固醇激素
6. **胰岛素抵抗/肥胖** - 降低SHBG，相对提高FAI

**男性常见原因：**
1. **低SHBG状态**：肥胖、2型糖尿病、甲状腺功能减退、肾病综合征
2. **外源性雄激素补充过量**
3. **肾上腺或睾丸分泌雄激素肿瘤**（罕见）

### ▶ FAI 降低（低雄激素状态）

**男性常见原因：**
1. **性腺功能减退**：
   - 原发性（睾丸本身问题）：克氏综合征、隐睾、化疗/放疗损伤
   - 继发性（垂体/下丘脑问题）：垂体瘤、高泌乳素血症、Kallmann综合征
2. **高SHBG状态**：肝硬化、甲状腺功能亢进、老年、HIV感染
3. **慢性消耗性疾病**：肾病、严重营养不良、厌食症
4. **药物**：雌激素、抗雄激素药物、阿片类药物、糖皮质激素

**女性常见原因：**
1. **肾上腺功能减退**（Addison病）
2. **垂体功能减退**
3. **严重营养不良或过度运动**（女运动员三联征）

## 三、异常后的建议处理方案

### 🔍 **第一步：复查与确认**
- **2-4周后复查**：排除生理性波动（应激、急性疾病、月经周期影响）
- **完善相关激素检测**：
  - 女性：DHEAS、雄烯二酮、17-羟孕酮（排除CAH）、LH/FSH、PRL、甲状腺功能
  - 男性：LH、FSH、PRL、雌二醇（排除芳香化酶活性过高）

### 🏥 **第二步：针对性检查**

**若FAI升高（尤其女性伴多毛/痤疮/月经稀发）：**
- **超声检查**：卵巢多囊样改变、肾上腺占位
- **糖代谢评估**：空腹胰岛素、OGTT（排查胰岛素抵抗）
- **皮质醇节律**：排除库欣综合征

**若FAI降低（尤其男性伴乏力/性欲减退）：**
- **垂体MRI**：排除垂体微腺瘤
- **骨密度检测**：长期低雄激素易导致骨质疏松
- **肝功能/甲状腺功能**：排查SHBG异常升高原因

### 🌿 **第三步：生活方式干预**

**针对FAI过高（特别是PCOS倾向）：**
1. **体重管理**：减重5-10%可显著降低雄激素水平，恢复排卵
2. **饮食调整**：低升糖指数（低GI）饮食，减少精制碳水，增加膳食纤维
3. **运动处方**：每周150分钟中等强度有氧运动+抗阻训练
4. **补充剂**：肌醇（Inositol）可能改善胰岛素敏感性

**针对FAI过低（特别是男性更年期或功能减退）：**
1. **睡眠优化**：保证7-8小时深度睡眠（睾酮主要在深睡眠期分泌）
2. **力量训练**：深蹲、硬拉等大肌群训练刺激睾酮分泌
3. **营养支持**：充足锌（牡蛎、瘦肉）、维生素D（日晒/补充剂）、健康脂肪（胆固醇是激素合成原料）
4. **压力管理**：慢性压力升高皮质醇，抑制睾酮合成

### 💊 **第四步：医学干预（需专科医生指导）**

- **FAI过高**：短效避孕药（调节月经、降低雄激素）、二甲双胍（改善胰岛素抵抗）、螺内酯（抗雄激素）
- **FAI过低**：睾酮替代治疗（TRT，严格掌握适应症）、HCG注射（用于继发性性腺功能减退）、治疗原发病（如垂体瘤手术）

---

## ⚠️ 重要提示

1. **FAI的局限性**：当SHBG水平极端异常（如严重肝病、甲亢）时，FAI估算可能失真，此时建议直接检测**游离睾酮**（透析法或质谱法）。
2. **性别差异**：女性轻度FAI升高非常常见（尤其育龄期），需结合临床表现（痤疮、多毛、脱发、月经）综合判断，而非仅凭数值。
3. **动态观察**：单次轻度异常不必过度焦虑，建议3个月后复查，观察趋势比单次数值更重要。
4. **避免自行用药**：切勿自行服用雄激素补充剂或抗雄激素药物，可能干扰内分泌轴，导致不可逆后果。

**建议就诊科室**：内分泌科（首选）、妇科内分泌（女性月经异常）、男科（男性性功能障碍）。
