**血汞（Mercury, Blood）** 是评估人体近期汞暴露负荷的重要生物标志物，主要反映**近期（数天至数周）**内进入血液循环的汞含量，包括无机汞和有机汞（主要是甲基汞）。

---

### 一、指标基本涵义

| 项目 | 说明 |
|------|------|
| **检测对象** | 全血或血清中的总汞含量（Total Blood Mercury） |
| **反映时段** | 近期暴露（有机汞半衰期约50-70天，无机汞约30-60天） |
| **主要形态** | - **有机汞（甲基汞）**：占血汞的90%以上，主要来自大型海鱼<br>- **无机汞**：来自职业暴露、含汞药物/化妆品、牙科填充物 |
| **参考范围** | - **正常**：<10 μg/L（或<50 nmol/L，不同实验室略有差异）<br>- **安全上限**：美国EPA建议<5.8 μg/L（育龄妇女及儿童）<br>- **中毒水平**：通常>100 μg/L（需结合临床症状） |

**注意**：血汞对**甲基汞暴露**最敏感，但对无机汞（如职业暴露）的反映不如**尿汞**准确。

---

### 二、常见异常及可能原因

#### 🔴 **升高（最常见异常）**

**1. 饮食因素（最常见）**
- **高汞鱼类摄入过量**：金枪鱼（尤其是长鳍金枪鱼/蓝鳍金枪鱼）、鲨鱼、剑鱼、鲭鱼（国王鲭）、马林鱼、方头鱼、旗鱼等大型掠食性鱼类
- **食用频率**：每周>3次大型深海鱼，或单次大量食用（如生鱼片聚餐）

**2. 职业/环境暴露**
- **职业**：牙医（处理汞合金）、氯碱工业工人、温度计/血压计制造、金矿开采、汞冶炼
- **环境**：居住于工业污染区（如既往水俣病地区）、吸入汞蒸气（打碎温度计/节能灯后处理不当）

**3. 医源性与生活接触**
- 含汞美白化妆品（某些非法添加汞的祛斑产品）
- 含汞传统药物（某些中药偏方）
- 牙科银汞合金（充填物），但通常仅引起轻度升高

**4. 特殊人群**
- 孕妇（甲基汞可通过胎盘屏障，影响胎儿神经发育）
- 长期严格素食者突然改为高鱼饮食

#### 🟢 **降低**
- **临床意义极小**：通常无需关注，可能提示近期蛋白质摄入不足或检测误差。

---

### 三、异常对应的建议处理

#### **轻度升高（10-50 μg/L，无症状）**

**立即行动：**
1. **饮食调整（关键）**：
   - **暂停高汞鱼类**：至少3-6个月内避免上述大型掠食性鱼类
   - **替代选择**：改吃低汞鱼类（三文鱼、鳕鱼、罗非鱼、虾、沙丁鱼、鲱鱼），每周2-3次，每次100-150g
   - **增加膳食纤维**：硒（巴西坚果、鸡蛋）和锌可拮抗汞毒性

2. **排查暴露源**：
   - 检查化妆品成分表（避免含"Mercury"、"Mercurio"、"汞"、"升汞"字样）
   - 回顾工作环境（是否接触温度计、血压计、荧光灯管破损）

3. **复查**：
   - 3个月后复查血汞，观察下降趋势

#### **中度升高（50-100 μg/L，或出现轻微症状）**

**医学干预：**
1. **进一步检查**：
   - **尿汞检测**：鉴别无机汞暴露（24小时尿汞更准）
   - **头发汞检测**：反映过去3-6个月甲基汞暴露水平（慢性暴露指标）
   - **肾功能**：尿β2-微球蛋白、尿NAG（早期肾损伤标志）
   - **神经功能检查**：手指震颤测试、记忆力评估、周围神经传导速度

2. **症状监测**：
   - 注意手抖、情绪波动、失眠、牙龈炎、金属味觉等症状

#### **重度升高（>100 μg/L，或出现中毒症状）**

**紧急处理：**
1. **立即脱离暴露源**：职业暴露者需调岗
2. **专科就诊**：中毒科/职业病科/肾内科
3. **驱汞治疗（螯合疗法）**：
   - 二巯丙磺钠（DMPS）
   - 二巯丁二酸（DMSA）
   - **注意**：孕妇禁用螯合剂，需权衡利弊
4. **器官保护**：监测肾功能、肝功能

---

### 四、特殊人群注意事项

| 人群 | 建议 |
|------|------|
| **育龄女性/孕妇** | 血汞>5.8 μg/L即需干预，甲基汞可致胎儿脑发育异常（Minamata病），建议孕前筛查 |
| **哺乳期妇女** | 甲基汞可经乳汁分泌，高汞时应暂停哺乳 |
| **儿童** | 神经系统对汞更敏感，血汞>5 μg/L需警惕 |

---

### 五、预防建议

1. **鱼类选择原则**：遵循"**小、浅、短**"——体型小、浅海鱼、生命周期短的鱼类汞含量低
2. **烹饪方式**：蒸煮可减少部分脂肪中的汞，但对甲基汞（与蛋白质结合）去除有限
3. **职业防护**：接触汞作业需佩戴防毒面具、定期轮岗、班后淋浴更衣
4. **家庭防护**：打碎水银温度计后，立即撒硫磺粉或用硬纸片收集，严禁用吸尘器（会气化扩散）

**⚠️ 重要提示**：血汞结果需结合临床表现和暴露史综合判断。一次轻度升高无需恐慌，但持续升高或伴有神经系统症状时，务必及时就诊职业病防治院或三甲医院中毒科。
