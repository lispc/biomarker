**DHEA-S（硫酸脱氢表雄酮，Dehydroepiandrosterone Sulfate）** 是体检中评估肾上腺功能和雄激素状态的核心指标之一。以下从临床意义、异常解读到处理建议进行系统说明：

---

### 一、指标基础涵义

**1. 本质与来源**
- **化学本质**：一种弱雄激素（肾上腺雄激素）的硫酸酯形式，是DHEA的代谢产物
- **分泌来源**：**95%以上来自肾上腺皮质网状带**（因此是肾上腺功能的特异性标志物），卵巢/睾丸分泌极少
- **生理作用**：
  - 作为"激素原"：可在外周组织转化为活性雄激素（睾酮）和雌激素（雌二醇、雌酮）
  - 调节免疫、抗皮质醇的分解代谢作用
  - 与衰老、骨密度、认知功能相关

**2. 检测优势**
- **稳定性高**：半衰期约10-12小时（比游离DHEA长），**无显著昼夜节律**，单次检测即可反映基础水平
- **不受性激素结合球蛋白（SHBG）影响**：比游离睾酮更能真实反映肾上腺雄激素活性

**3. 参考范围**（各实验室略有差异，且与年龄强相关）：
- **成年男性**：2.0–10.0 mg/L（或5.0–27.0 μmol/L）
- **成年女性**：1.5–8.5 mg/L（或4.0–22.0 μmol/L）
- **随年龄变化**：20-30岁达峰值，之后每十年下降约20%，60岁后可能降至峰值的20-30%（"肾上腺功能暂停"）

---

### 二、常见异常及可能原因

#### **A. DHEA-S 升高（高肾上腺雄激素血症）**

| 临床情境 | 可能机制/疾病 | 典型伴随表现 |
|---------|-------------|------------|
| **肾上腺皮质增生/肿瘤** | 先天性肾上腺皮质增生（CAH，尤其是21-羟化酶缺乏症）、肾上腺腺瘤/癌 | 女性男性化（声音低沉、多毛）、高血压、低钾血症（癌） |
| **多囊卵巢综合征（PCOS）** | 约30-40%患者伴轻度升高（肾上腺来源雄激素过多） | 月经不调、痤疮、黑棘皮症、超声见多囊卵巢 |
| **多毛症/女性脱发** | 肾上腺雄激素对毛囊刺激 | Ferriman-Gallwey评分升高、顶秃 |
| **药物/补剂** | 服用DHEA保健品、孕烯醇酮、某些抗癫痫药 | 用药史明确 |
| **特发性多毛症** | 外周组织对雄激素敏感性增高，但DHEA-S仅轻度升高 | 家族史阳性 |
| **应激/ACTH刺激** | 急性生理/心理应激（暂时性） | 皮质醇同步升高 |

#### **B. DHEA-S 降低（肾上腺雄激素不足）**

| 临床情境 | 可能机制/疾病 | 典型伴随表现 |
|---------|-------------|------------|
| **原发性肾上腺功能不全** | 艾迪生病（Addison病）、自身免疫性肾上腺炎 | 疲劳、皮肤色素沉着、低钠高钾、低血压 |
| **继发性肾上腺功能不全** | 垂体ACTH分泌不足（席汉综合征、垂体瘤术后、长期糖皮质激素抑制） | 低血糖、苍白、继发性甲减、无色素沉着 |
| **衰老（生理性）** | 肾上腺网状带萎缩 | 肌肉量减少、骨密度下降、认知减退（非特异性） |
| **慢性疾病消耗** | 严重肝病（合成减少）、肾病综合征（排泄异常）、癌症恶病质、HIV/AIDS | 原发病表现突出 |
| **药物抑制** | 长期糖皮质激素（泼尼松等）、酮康唑、阿片类药物 | 用药史明确 |
| **营养不良** | 厌食症、过度节食、极端运动 | BMI过低、闭经（女） |

### 三、异常后的建议处理流程

#### **第一步：复查与确认（排除干扰）**
- **重复检测**：排除实验室误差或急性应激（如抽血前剧烈运动、熬夜、急性感染）
- **检测时间**：无需空腹，但建议上午检测（虽无昼夜节律，但标准化）
- **药物史回顾**：停用DHEA补剂、生物素（高剂量干扰检测）至少1周

#### **第二步：分层处理策略**

**情况1：DHEA-S显著升高（>2倍上限，尤其女性）**
- **必查项目**：
  - **ACTH刺激试验**：鉴别肾上腺 vs 垂体来源
  - **肾上腺CT/MRI**：排查腺瘤/增生（尤其伴皮质醇或醛固酮异常时）
  - **17-羟孕酮（17-OHP）**：筛查先天性肾上腺皮质增生（CAH）
  - **睾酮、雄烯二酮、LH/FSH**：鉴别PCOS（睾酮可能正常但DHEA-S高提示肾上腺型PCOS）
- **专科转诊**：内分泌科（肾上腺专科）或妇科内分泌

**情况2：DHEA-S轻度升高（1.5-2倍上限，女性）**
- **排查PCOS**：超声+糖耐量试验+胰岛素释放
- **生活方式干预**（见下文）3个月后复查
- **皮肤科评估**：若主要表现为多毛/痤疮，可考虑螺内酯（抗雄激素）治疗（需医生处方）

**情况3：DHEA-S降低（低于年龄匹配下限）**
- **必查项目**：
  - **晨间皮质醇+ACTH**：判断原发/继发性肾上腺功能不全
  - **肾功能+肝功能**：排除慢性疾病影响
  - **甲状腺功能**：甲减常伴随DHEA-S低
- **功能医学评估**：若仅为轻度降低伴疲劳、性欲减退，需评估"肾上腺疲劳"（医学界有争议，但需排除慢性疲劳综合征）

#### **第三步：生活方式与干预**

**针对升高的管理**：
1. **压力管理**：冥想、正念减压（降低ACTH刺激）
2. **体重控制**：减重5-10%可显著降低PCOS患者的DHEA-S
3. **饮食调整**：
   - 低升糖指数饮食（减少胰岛素抵抗，间接降低肾上腺雄激素）
   - 避免甘草（拟醛固酮作用可能刺激肾上腺）
4. **补剂谨慎**：绝对避免自行服用DHEA补剂（除非明确缺乏且医生指导）

**针对降低的管理**：
1. **营养支持**：
   - 充足的胆固醇摄入（激素合成原料）：蛋黄、深海鱼
   - 维生素D3（与DHEA-S正相关，建议维持>30 ng/mL）
   - 孕烯醇酮（Pregnenolone）补剂（仅在医生指导下，可能改善轻度不足）
2. **运动调节**：过度有氧运动可能进一步降低DHEA-S，建议改为抗阻训练+适度有氧
3. **睡眠优化**：保证深度睡眠（肾上腺修复在夜间）
4. **替代治疗**：仅用于确诊肾上腺功能不全者（氢化可的松+DHEA替代，需严格监测）

---

### 四、特别注意事项

1. **年龄解读**：老年人轻度降低多为生理性，无需过度干预；年轻人显著降低需警惕早发性肾上腺功能不全。
2. **性别差异**：女性DHEA-S升高更易导致临床症状（男性本底睾酮高，DHEA-S轻度升高常被掩盖）。
3. **与DHEA的区别**：血清DHEA波动大，DHEA-S才是稳定指标，不要混淆。
4. **肿瘤标志物**：DHEA-S极高（>15 mg/L）伴雄烯二酮升高，强烈提示肾上腺癌，需紧急影像学检查。

**重要提示**：DHEA-S异常通常是内分泌系统失衡的信号，**不建议自行用药（尤其是DHEA保健品）**。应在内分泌科医生指导下，结合临床表现、影像学及其他激素检查综合判断。
