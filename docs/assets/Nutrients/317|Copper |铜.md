在常规体检或专科检查中，**铜（Copper）**通常以**血清铜（Serum Copper）**、**铜蓝蛋白（Ceruloplasmin）**或**24小时尿铜（Urinary Copper）**的形式检测。它是人体必需的微量元素，但代谢异常可能导致严重疾病。

---

### 一、指标涵义与生理作用

铜是多种关键酶的辅因子，主要功能包括：
- **铁代谢**：铜蓝蛋白（含90%的血清铜）协助铁从组织转运至骨髓，缺铜会导致类似缺铁性贫血。
- **抗氧化**：参与超氧化物歧化酶（SOD）构成，清除自由基。
- **结缔组织合成**：参与胶原与弹性蛋白交联（赖氨酰氧化酶）。
- **神经与免疫**：参与神经髓鞘形成和免疫功能。

**常见检测组合**：
- **血清铜**：反映循环中铜总量（正常约 **70–140 μg/dL** 或 **11–22 μmol/L**）。
- **铜蓝蛋白**：铜的主要运输蛋白（正常约 **20–40 mg/dL**）。
- **游离铜（非铜蓝蛋白结合铜）**：计算公式为 *血清铜(μg/dL) – 3.15 × 铜蓝蛋白(mg/dL)*，正常<15–20 μg/dL。此指标对**肝豆状核变性（Wilson病）**诊断最关键。
- **24小时尿铜**：反映铜排泄量（正常<40 μg/24h）。

### 二、常见异常及可能原因

#### 1. 血清铜升高（高铜血症）
| 可能原因 | 机制说明 |
|---------|---------|
| **急性期反应** | 炎症、感染、创伤、心梗时，铜蓝蛋白作为急性期蛋白升高。 |
| **雌激素影响** | 妊娠、口服避孕药、激素替代治疗（HRT）可刺激铜蓝蛋白合成。 |
| **胆汁淤积** | 胆道梗阻（原发性胆汁性胆管炎等）导致铜经胆汁排泄受阻。 |
| **血液系统疾病** | 缺铁性贫血、巨幼细胞性贫血、白血病（铜利用障碍或释放增加）。 |
| **恶性肿瘤** | 淋巴瘤、霍奇金病（副肿瘤综合征）。 |
| **甲状腺功能亢进** | 甲状腺激素促进铜蓝蛋白合成。 |

#### 2. 血清铜降低（低铜血症）
| 可能原因 | 机制说明 |
|---------|---------|
| **Wilson病（肝豆状核变性）** | 遗传病（ATP7B基因突变），铜蓝蛋白合成障碍，铜沉积在肝、脑、角膜。血清总铜常正常或偏低，但**游离铜显著升高**（毒性铜）。 |
| **Menkes病（门克斯病）** | 婴幼儿X连锁遗传病，铜吸收障碍，导致严重铜缺乏，发育迟缓。 |
| **吸收不良** | 严重营养不良、乳糜泻、短肠综合征、长期腹泻。 |
| **肾病综合征** | 铜蓝蛋白从尿中大量丢失。 |
| **长期肠外营养** | 未补充铜制剂。 |
| **铜蓝蛋白缺乏症** | 罕见遗传病（如aceruloplasminemia）。 |
| **过量补锌** | 锌竞争性抑制肠道铜吸收（如长期每日补锌>150mg）。 |

#### 3. 尿铜异常
- **尿铜升高**：Wilson病（常>100 μg/24h）、胆汁淤积、铜中毒。
- **尿铜降低**：铜缺乏状态、Menkes病。

### 三、异常时的后续处理建议

#### 第一步：复查与鉴别
- **排除干扰**：确认未在急性感染期、妊娠期检测；停用含雌激素药物或高剂量维生素/矿物质补充剂（含铜或锌）2–4周后复查。
- **联合解读**：必须结合**铜蓝蛋白**、**肝功能**、**血常规**、**炎症指标（CRP/ESR）**综合判断。

#### 第二步：针对性检查

**若怀疑Wilson病（金标准排查）**：
- **眼科检查**：裂隙灯查**Kayser-Fleischer环**（角膜铜沉积，金棕色环）。
- **尿铜检测**：24小时尿铜（Wilson病患者通常升高）。
- **肝活检+肝铜定量**：肝铜含量>250 μg/g干重（正常<50）可确诊。
- **基因检测**：ATP7B基因突变分析（确诊及家系筛查）。
- **神经科评估**：脑MRI（基底节异常）、神经系统查体（震颤、肌张力障碍）。

**若怀疑铜缺乏**：
- 营养评估（饮食史、体重变化）。
- 粪便检查（脂肪泻提示吸收不良）。
- 血清锌检测（排除锌过量拮抗）。
- 骨密度检查（长期缺铜可能导致骨质疏松）。

**若怀疑胆汁淤积**：
- 腹部超声/MRCP（磁共振胰胆管成像）。
- 自身抗体检测（AMA-M2等）。

#### 第三步：干预措施

**生活方式调整**：
- **高铜血症（非Wilson病）**：
  - 限制高铜食物：动物肝脏、贝类（牡蛎、龙虾）、坚果（腰果、核桃）、黑巧克力、蘑菇、豆类。
  - 避免使用铜制炊具/水管（尤其酸性食物）。
- **低铜血症**：
  - 增加铜来源：贝类、牛肝、芝麻、黑巧克力、全谷物、豆类。
  - 避免过量补锌（每日锌摄入建议<40mg）。
- **Wilson病患者**：
  - 需终身**低铜饮食**（每日铜摄入<1mg），严格避免上述高铜食物及饮用水铜超标。

**医学治疗**：
- **Wilson病**：驱铜治疗（D-青霉胺、曲恩汀、锌制剂），终身服药。
- **铜缺乏**：口服铜补充剂（硫酸铜或葡萄糖酸铜），通常1–2mg/日，需在医生指导下进行（过量可致肝损伤）。
- **继发于其他疾病**：治疗原发病（如解除胆道梗阻、改善营养状态）。

---

### 四、特别提醒
- **Wilson病是铜代谢异常中最危急的情况**，青少年或青年出现不明原因肝病、神经精神症状（抑郁、震颤）或溶血性贫血时，必须排查此病。
- **单一血清铜指标意义有限**，必须结合铜蓝蛋白计算**游离铜**水平，或进行尿铜检测。
- **参考值因实验室方法而异**，请以报告单标注范围为准。
