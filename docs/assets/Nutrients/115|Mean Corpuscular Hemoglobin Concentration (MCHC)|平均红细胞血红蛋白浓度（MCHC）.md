**Mean Corpuscular Hemoglobin Concentration（MCHC，平均红细胞血红蛋白浓度）** 是血常规中反映红细胞内血红蛋白充盈程度的核心指标，常用于贫血的形态学分类。

---

### 一、基本涵义与参考范围

**定义**  
MCHC 指**每升红细胞中所含血红蛋白的平均克数**（或每100ml红细胞压积中的血红蛋白克数），反映红细胞内血红蛋白的浓缩程度。

**计算公式**  
$$MCHC (g/L) = \frac{Hb (g/L)}{Hct (\%)} \times 100$$  
或 $$MCHC = \frac{Hb}{RBC} \times MCV$$

**参考范围**（因实验室而异，大致）：  
- **320–360 g/L**（或 32–36 g/dL）  
- 新生儿略高（约 330–370 g/L）

**临床意义**：  
- **低 MCHC**：提示红细胞"苍白"、血红蛋白填充不足（低色素性贫血）  
- **高 MCHC**：提示红细胞内血红蛋白高度浓缩（高色素性），但真性升高较少见，多数为假性升高

---

### 二、常见异常及可能原因

#### 1. MCHC 降低（< 320 g/L）——最常见
**提示低色素性贫血**，红细胞"颜色浅、内含少"：

| 可能原因 | 特征与机制 |
|---------|-----------|
| **缺铁性贫血** | 最常见原因。铁不足导致血红蛋白合成障碍，早期可能正常，进展期明显降低。常伴 MCV↓、MCH↓、RDW↑。 |
| **地中海贫血**（珠蛋白生成障碍） | 珠蛋白链合成障碍，靶形红细胞多见。常为**小细胞低色素**（MCV↓↓，MCHC↓或正常低限），但 RDW 通常正常或轻度升高。 |
| **慢性病性贫血** | 炎症/肿瘤导致铁利用障碍，多为正细胞性，部分可表现为低色素。 |
| **铁粒幼细胞性贫血** | 铁利用障碍，环形铁粒幼细胞，可伴 MCV 正常或增高（双相红细胞群）。 |
| **铅中毒** | 抑制血红素合成酶，干扰血红蛋白合成。 |
| **血红蛋白 E 病** | 血红蛋白变异体，不稳定。 |

#### 2. MCHC 升高（> 360 g/L）
**真性升高罕见**，需首先排除实验室干扰：

**真性升高（病理）**：
- **遗传性球形红细胞增多症（HS）**：红细胞膜缺陷导致球形变，细胞脱水浓缩，MCHC 常 > 360 g/L（可达 380–400+），伴 MCV 正常或轻度降低，外周血涂片见球形红细胞。
- **自身免疫性溶血性贫血（冷凝集素病）**：红细胞凝集可致假性升高，但部分病例因红细胞损伤也可出现真性改变。

**假性升高（技术性，更常见）**：
- **样本溶血**：红细胞破裂释放血红蛋白，检测时误算为"细胞内"血红蛋白。
- **高脂血症/乳糜血**：血浆浊度干扰光学检测。
- **白细胞极度增高**（>100×10⁹/L）：白细胞体积干扰红细胞通道检测。
- **冷凝集素**：红细胞在体外凝集，被计数为"大细胞"导致计算误差。
- **红细胞内包涵体**（如 Heinz 小体）：极端情况下。

### 三、后续处理建议

#### 第一步：复核与验证（排除假性异常）
- **复查血常规**：排除样本溶血、脂血或冷凝集干扰（37℃温育后复查）。
- **观察相关指标**：
  - 若 MCHC > 380 g/L，几乎肯定为假性升高（仪器误差或溶血）。
  - 结合 **MCV、MCH、RDW** 判断贫血类型（小细胞低色素？正细胞？）。

#### 第二步：针对性检查

**对于 MCHC 降低（低色素性）**：
1. **铁代谢全套**：血清铁、铁蛋白（最敏感）、总铁结合力（TIBC）、转铁蛋白饱和度。  
   - 缺铁：铁蛋白↓，TIBC↑，饱和度↓  
   - 地中海贫血：铁蛋白正常或偏高（易合并铁过载）
2. **血红蛋白电泳**：筛查地中海贫血（HbA2↑）或血红蛋白病（HbE、HbS 等）。
3. **基因检测**：确诊 α/β 地中海贫血。
4. **网织红细胞计数**：评估骨髓造血代偿情况。
5. **骨髓穿刺**（必要时）：排除铁粒幼细胞性贫血、骨髓病性贫血。

**对于 MCHC 升高**：
1. **外周血涂片镜检**：寻找球形红细胞（HS）、裂红细胞（微血管病）。
2. **红细胞渗透脆性试验**：HS 患者脆性增加。
3. **库姆试验（Coombs Test）**：排查自身免疫性溶血。
4. **红细胞膜蛋白电泳**：确诊遗传性球形红细胞增多症。

#### 第三步：生活方式与干预

**如果是缺铁性贫血**：
- **饮食调整**：增加红肉、动物肝脏、血制品、贝类（牡蛎）、深绿色蔬菜（菠菜、芥蓝）。
- **促进吸收**：搭配维生素 C（柑橘、猕猴桃）同食；避免餐时饮茶、咖啡（单宁酸抑制铁吸收）。
- **药物**：口服硫酸亚铁、富马酸亚铁等（需医师处方，通常 3–6 个月补足储存铁）。

**如果是地中海贫血**：
- **禁止盲目补铁**：除非确诊合并缺铁，否则补铁可能导致铁过载损伤心肝。
- **遗传咨询**：轻型患者需婚前/孕前筛查，避免与同类基因携带者婚配（预防重型患儿）。
- **避免氧化性药物**：如磺胺类、伯氨喹（可能诱发溶血）。

**如果是遗传性球形红细胞增多症**：
- 脾切除是根治手段（通常 5 岁后考虑），术后需预防肺炎链球菌感染（疫苗接种）。

---

### 四、特别提醒

1. **不要孤立看 MCHC**：必须结合 **MCV（红细胞平均体积）** 和 **MCH（平均血红蛋白量）**：
   - MCV↓ + MCHC↓ = 小细胞低色素（缺铁/地贫）
   - MCV正常 + MCHC正常 = 正色素（急性失血、慢性病）
   - MCV↑ + MCHC正常/↓ = 大细胞（巨幼细胞贫血，MCHC 通常不低，但 MCV 很高）

2. **轻度异常的处理**：若仅 MCHC 轻度偏离（如 315 或 365），而 Hb、RBC 正常，无临床症状，可能为个体变异或轻度缺铁早期，建议 1–3 个月后复查，无需过度焦虑。

3. **新生儿与孕妇**：新生儿 MCHC 生理性偏高；孕妇因血液稀释，MCHC 可轻度降低，需结合孕期标准判断。

**建议**：拿到异常报告后，首诊**血液科**或**内科**，由医生根据完整血常规（含红细胞体积分布宽度 RDW）、病史（月经量、饮食习惯、家族史）决定进一步检查方向。
