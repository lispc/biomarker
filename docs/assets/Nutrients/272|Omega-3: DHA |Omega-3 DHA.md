**Omega-3: DHA（Docosahexaenoic Acid，二十二碳六烯酸）** 是Omega-3多不饱和脂肪酸家族中最重要的成员之一。以下从临床角度系统解析该指标：

---

### 一、指标涵义

**1. 生物学定义**
- DHA是一种长链多不饱和脂肪酸（C22:6, n-3），人体自身合成能力极弱（转化率<5%），需依赖膳食摄入
- 检测形式：通常以**红细胞膜DHA含量**（Omega-3指数的组成部分）或**血浆磷脂DHA百分比**表示

**2. 生理功能**
- **神经发育**：大脑皮层和视网膜的主要结构脂肪（占大脑脂肪酸25%）
- **心血管保护**：降低甘油三酯、抗炎、稳定细胞膜、改善血管内皮功能
- **抗炎调节**：生成消退素（Resolvins）和保护素（Protectins），主动消退炎症

**3. 正常参考范围**（因检测方法而异）
- **Omega-3指数**（EPA+DHA）：**>8%为理想**，4-8%为中等风险，<4%为高风险（心血管事件风险增加）
- **单独DHA**：通常占血浆总脂肪酸的**1.5-3.5%**（不同实验室标准不同）

---

### 二、常见异常及原因分析

#### ▶ **DHA偏低（最常见）**

| 可能原因 | 具体机制/表现 |
|---------|--------------|
| **膳食摄入不足** | 很少吃深海鱼（三文鱼、鲭鱼、沙丁鱼、秋刀鱼），依赖陆地动物蛋白 |
| **素食/纯素食** | 植物中仅含ALA（亚麻酸），转化为DHA效率极低（<1%） |
| **脂肪吸收障碍** | 慢性胰腺炎、胆囊切除、乳糜泻、克罗恩病导致脂溶性营养素吸收不良 |
| **代谢竞争** | 过量摄入Omega-6（玉米油、葵花籽油、加工食品），n-6/n-3比例失衡（理想比例<4:1，现代人常达15-20:1） |
| **生活方式** | 吸烟（氧化应激消耗DHA）、过量饮酒、高糖饮食抑制合成 |
| **药物影响** | 某些降脂药（如考来烯胺）、激素药物、抗癫痫药 |
| **遗传因素** | FADS1/FADS2基因多态性（脂肪酸去饱和酶活性低，转化能力差） |

#### ▶ **DHA偏高（较少见）**

| 可能原因 | 具体机制/表现 |
|---------|--------------|
| **过量补充** | 长期超量服用鱼油/藻油（>3g/天），或短期内大量进食深海鱼 |
| **代谢异常** | 过氧化物酶体疾病（罕见遗传病，DHA合成障碍但检测值可能异常分布） |
| **检测干扰** | 采血前24小时内服用大剂量补充剂（急性升高） |

---

### 三、异常后的处理建议

#### ▶ **DHA偏低的干预方案**

**1. 膳食优化（首选）**
- **增量策略**：每周至少2-3次高脂深海鱼（每次100-150g），优选低汞品种（三文鱼、鲭鱼、沙丁鱼、鲱鱼）
- **素食方案**：藻类DHA补充剂（藻油），避免依赖亚麻籽/核桃（转化太低）
- **比例调整**：减少精炼植物油（玉米油、葵花籽油），改用橄榄油、茶油；限制加工食品（富含Omega-6）

**2. 营养补充剂**
- **常规剂量**：一般建议250-500mg DHA+EPA/天（心血管一级预防）
- **高需求人群**：孕妇/哺乳期（200-300mg DHA/天）、心血管疾病患者（需医生指导，可达1-2g/天）
- **形式选择**：甘油三酯型（rTG）吸收率优于乙酯型（EE）；藻油适合素食者

**3. 医学排查**
- 如调整饮食后仍低，建议检查：
  - **消化系统**：粪便脂肪球、脂溶性维生素（A、D、E、K）水平
  - **基因检测**：FADS基因多态性分析（明确代谢能力）
  - **炎症指标**：hs-CRP（评估炎症状态）

**4. 复查时机**
- 生活方式干预后**3-6个月**复查（红细胞膜脂肪酸更新周期约4个月）

#### ▶ **DHA偏高的处理**

- **减量调整**：暂停补充剂1-2个月，或改为隔日服用
- **凝血监测**：如>1000mg/天长期服用，需监测凝血功能（APTT、PT），特别是术前或服用抗凝药（华法林、阿司匹林）者
- **氧化应激检查**：极高剂量可能增加脂质过氧化，建议检查氧化低密度脂蛋白（ox-LDL）

---

### 四、特殊人群注意事项

| 人群 | 特殊建议 |
|------|---------|
| **孕妇/儿童** | DHA对胎儿脑眼发育关键，偏低需优先纠正，但避免高汞鱼类（剑鱼、鲨鱼） |
| **出血性疾病患者** | 高DHA可能延长出血时间，需在医生指导下调整 |
| **房颤患者** | 高剂量Omega-3（>4g/天）可能与房颤风险相关，需心内科评估 |
| **海鲜过敏者** | 选择藻油而非鱼油 |

### 五、总结
DHA是反映**膳食质量**和**心血管保护状态**的重要生物标志物。现代人普遍偏低，重点在于**增加深海鱼摄入**和**优化脂肪酸比例**（减少Omega-6）。如检测异常，建议先通过3个月饮食干预，无效时再考虑补充剂或深入检查吸收代谢功能。

*注：具体补充剂量和干预方案需结合总胆固醇、甘油三酯、凝血功能等指标，由医生或临床营养师个体化制定。*
