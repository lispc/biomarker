**总蛋白（Protein, Total, TP）** 是临床生化检查中最基础的指标之一，反映的是血液中所有蛋白质的总浓度，主要由**白蛋白（Albumin，约占60%）**和**球蛋白（Globulin，约占40%）**组成。

---

### 一、指标涵义与参考范围

**生理意义：**
- **维持胶体渗透压**：防止水分从血管渗出到组织间隙（水肿）
- **运输功能**：携带激素、药物、脂肪酸、钙等物质
- **免疫功能**：球蛋白包含免疫抗体
- **营养状态标志**：反映肝脏合成功能和蛋白质摄入/丢失平衡

**参考范围（成人，血清/血浆）：**
- **65–85 g/L**（或 6.5–8.5 g/dL）
- *注：不同实验室方法（双缩脲法）可能略有差异，孕期可能轻度降低*

---

### 二、常见异常及可能原因

#### 🔺 **总蛋白升高（>85 g/L，高蛋白血症）**

| 类型 | 常见原因 | 机制说明 |
|------|---------|---------|
| **相对性升高**（最常见） | **脱水/血液浓缩** | 呕吐、腹泻、烧伤、饮水不足、利尿剂使用导致血浆水分减少，蛋白浓度相对升高 |
| **绝对性升高** | **多发性骨髓瘤**等浆细胞病 | 异常浆细胞大量分泌单克隆免疫球蛋白（M蛋白） |
| | **慢性炎症/感染** | 结核、慢性肝炎、自身免疫病导致球蛋白（免疫球蛋白）反应性增多 |
| | **自身免疫病** | 系统性红斑狼疮、类风湿关节炎 |
| | **肝硬化（特殊类型）** | 球蛋白显著升高，白蛋白降低，但总和可能正常或偏高 |

#### 🔻 **总蛋白降低（<65 g/L，低蛋白血症）**

| 类型 | 常见原因 | 关键线索 |
|------|---------|---------|
| **合成不足** | **肝硬化、重症肝炎** | 白蛋白仅由肝脏合成，肝功能受损时显著下降；常伴腹水、黄疸 |
| | **营养不良** | 长期饥饿、极端节食、素食蛋白摄入不足、老年人吸收不良 |
| **丢失过多** | **肾病综合征** | 肾小球滤过屏障破坏，大量蛋白尿（>3.5g/24h），伴严重水肿 |
| | **蛋白丢失性肠病** | 肠道炎症（如克罗恩病）、淋巴管扩张导致蛋白从肠道丢失 |
| | **烧伤/大出血** | 血浆直接丢失 |
| **消耗增加** | **恶性肿瘤、甲亢、结核** | 高代谢状态分解蛋白加速 |
| **稀释性** | **水潴留**（心衰、 SIADH） | 输液过多或水钠潴留导致血液稀释 |

### 三、后续处理建议

#### **第一步：复查与鉴别**
- **排除干扰因素**：脱水、剧烈运动、长时间站立（体位性蛋白波动）、溶血标本
- **计算 A/G 比值**（白蛋白/球蛋白）：正常 1.2–2.4:1
  - A/G 倒置（<1）：提示肝硬化、多发性骨髓瘤
- **查看分项**：单独查白蛋白、球蛋白、蛋白电泳

#### **第二步：针对性检查**

**若升高（怀疑球蛋白增多）：**
- **血清蛋白电泳（SPEP）**：区分多克隆（炎症） vs 单克隆（骨髓瘤）球蛋白增高
- **免疫固定电泳**：确诊 M 蛋白类型
- **骨髓穿刺**：排除血液系统肿瘤
- **炎症指标**：CRP、血沉、自身抗体（ANA、RF）

**若降低（怀疑合成不足或丢失）：**
- **肝功能全套**（ALT、AST、胆红素、凝血功能）：评估肝脏合成功能
- **肾功能 + 尿常规 + 24小时尿蛋白**：排查肾病综合征
- **粪便 α-1 抗胰蛋白酶**：排查肠道蛋白丢失
- **腹部超声/CT**：检查肝脏形态、腹水、淋巴结

#### **第三步：生活方式与随访**

**针对高蛋白血症：**
- **脱水导致**：增加饮水，纠正电解质，治疗原发病（止泻、退热）
- **疾病导致**：需专科治疗（血液科、风湿免疫科），避免高嘌呤饮食（骨髓瘤患者）

**针对低蛋白血症：**

| 原因 | 饮食调整 | 医疗干预 |
|------|---------|---------|
| **营养不良** | 每日蛋白摄入 1.2–1.5g/kg 体重；优选优质蛋白（鸡蛋、鱼肉、瘦肉、乳清蛋白）；分餐制促进吸收 | 严重缺乏（<50g/L）需静脉输注人血白蛋白（短期应急） |
| **肝病** | 适量蛋白（避免诱发肝性脑病），植物蛋白为主；低盐饮食控制腹水 | 保肝治疗，病因治疗（抗病毒、戒酒） |
| **肾病** | **低蛋白饮食**（0.6–0.8g/kg）+ 必需氨基酸；优质蛋白占50%以上；严格限盐 | ACEI/ARB 减少蛋白尿，治疗原发肾病 |

**通用建议：**
- **3–6 个月后复查**：轻度异常（±5 g/L）无症状者，调整饮食后复查
- **动态监测**：慢性病患者每 3 个月检查一次，观察趋势比单次数值更重要
- **警惕危急值**：<40 g/L（严重水肿/休克风险）或 >100 g/L（高粘滞血症风险）需立即就医

### 四、特别提醒

1. **单项异常意义有限**：总蛋白需结合**白蛋白、前白蛋白、胆碱酯酶**评估肝脏合成功能；结合**肌酐、尿素氮**评估肾功能。
2. **妊娠期变化**：孕中晚期血容量增加可导致生理性总蛋白降低（约降低 10–15 g/L），属正常现象。
3. **假性升高**：使用大剂量青霉素、促蛋白合成类固醇可能干扰检测结果。

**建议**：拿到报告后先看 A/G 比值，再看白蛋白绝对值，最后结合临床症状（水肿、乏力、消瘦）决定是否需要深入检查。如有持续异常，建议挂**消化内科**（肝病）或**肾内科**（肾病）或**血液科**（球蛋白异常）进一步诊治。
