# Atherogenic Index of Plasma (AIP) / 血浆动脉粥样硬化指数

## 1. 基本定义与计算

**AIP** 是基于血脂水平计算得出的衍生指标，用于评估**动脉粥样硬化**和**心血管疾病（CVD）**风险。

### 计算公式
$$
\text{AIP} = \log_{10} \left( \frac{\text{TG}}{\text{HDL-C}} \right)
$$

- **TG**: 甘油三酯（Triglycerides）
- **HDL-C**: 高密度脂蛋白胆固醇（High-Density Lipoprotein Cholesterol）
- **单位要求**: TG 与 HDL-C 需使用相同单位（通常采用 mmol/L，若使用 mg/dL 计算结果数值相同）

> **物理意义**: 反映血液中致动脉粥样硬化颗粒（如残粒脂蛋白）与抗动脉粥样硬化颗粒（HDL）的平衡状态。数值越高，提示脂质代谢紊乱越严重，血管壁脂质沉积风险越大。

---

## 2. 参考范围与风险分层

| AIP 数值范围 | 风险等级 | 临床解读 |
|-------------|---------|---------|
| **< 0.1** | 低风险 | 脂质代谢相对健康，动脉粥样硬化风险较低 |
| **0.1 – 0.24** | 中等风险 | 需关注，建议定期监测 |
| **> 0.24** | 高风险 | 显著增加冠心病、脑卒中等心血管事件风险 |
| **> 0.35** | 极高风险 | 强烈提示严重胰岛素抵抗或代谢综合征 |

*注：不同实验室/研究可能略有差异，以上为临床常用标准*

---

## 3. 常见异常及原因分析

### 🔴 AIP 升高（主要临床关注点）

当 AIP > 0.24 时，提示**致动脉粥样硬化倾向增强**。

#### 常见原因：

**代谢性因素**
- **胰岛素抵抗/代谢综合征**: 最常见原因，TG 升高伴随 HDL-C 降低
- **2 型糖尿病**: 血糖控制不佳导致脂质代谢紊乱
- **肥胖（尤其腹型肥胖）**: 内脏脂肪过多促进游离脂肪酸释放

**生活方式因素**
- 高糖/高精制碳水化合物饮食
- 过量饮酒（显著升高 TG）
- 久坐不动、缺乏有氧运动
- 吸烟（降低 HDL-C）

**继发性/病理性因素**
- 甲状腺功能减退（甲减）
- 慢性肾病/肾病综合征
- 非酒精性脂肪性肝病（NAFLD）
- 某些药物：糖皮质激素、β-受体阻滞剂、噻嗪类利尿剂、雌激素治疗

**遗传因素**
- 家族性混合型高脂血症
- 脂蛋白脂酶（LPL）缺陷

### 🟢 AIP 降低（较少见，通常意义较小）

- **数值过低（如 < -0.5）**: 可能提示 HDL-C 极高或 TG 极低
- **可能原因**: 剧烈运动后、长期严格低脂饮食、甲状腺功能亢进（需结合其他指标判断）
- **临床意义**: 通常视为保护性状态，但极值需排除吸收不良或消耗性疾病

---

## 4. 异常后的处理建议

### 第一步：复核与确认（1-2 周内）
- **复查血脂四项**: 排除实验室误差（尤其 TG 受饮食影响大，需空腹 12-14 小时抽血）
- **确认计算准确性**: 确保 TG 与 HDL-C 单位统一

### 第二步：分层评估与检查

#### 基础评估（所有 AIP 升高者）
| 检查项目 | 目的 |
|---------|------|
| **空腹血糖 + HbA1c** | 筛查糖尿病/糖尿病前期 |
| **空腹胰岛素 + HOMA-IR** | 评估胰岛素抵抗程度 |
| **肝功能 + 肝脏超声** | 排查非酒精性脂肪肝 |
| **甲状腺功能（TSH、FT4）** | 排除甲减 |
| **肾功能（肌酐、尿素氮）** | 排除肾病继发改变 |
| **血压测量** | 评估代谢综合征其他组分 |

#### 进阶检查（中高风险或合并其他危险因素）
- **颈动脉超声**: 评估内膜中层厚度（IMT）及斑块形成
- **冠脉钙化评分（CAC）**: 评估冠状动脉粥样硬化负荷
- **载脂蛋白 B (ApoB)** 与 **小而密 LDL（sdLDL）**: 更精准评估致动脉粥样硬化颗粒数
- **超敏 C 反应蛋白（hs-CRP）**: 评估血管炎症状态

### 第三步：干预策略

#### 🥗 生活方式干预（基石治疗，持续 3-6 个月）

**饮食调整**
- **降低 TG**: 
  - 严格限制精制糖、果糖饮料、甜点（每日添加糖 < 25g）
  - 减少精制碳水，增加全谷物、杂豆、膳食纤维（每日 25-30g）
  - 限制酒精（尤其烈酒，最好戒酒）
- **提升 HDL-C**: 
  - 增加单不饱和脂肪酸（橄榄油、坚果、牛油果）
  - 摄入富含 Omega-3 脂肪酸的食物（深海鱼每周 2-3 次，或补充鱼油）
- **整体模式**: 推荐地中海饮食或 DASH 饮食

**运动处方**
- **有氧运动**: 每周 ≥150 分钟中等强度（快走、游泳、骑行），或 75 分钟高强度
- **抗阻训练**: 每周 2 次，增加肌肉量改善胰岛素敏感性
- **减重目标**: 体重下降 5-10% 可显著改善 AIP

**行为修正**
- 戒烟（可提升 HDL-C 10-20%）
- 睡眠管理（保证 7-8 小时，改善睡眠呼吸暂停）

#### 💊 医学干预（生活方式干预 3 个月无效，或初始风险极高）

**药物选择（需医生评估）**
- **他汀类药物**: 基础降脂，主要降低 LDL-C，轻度改善 AIP
- **贝特类药物（Fibrates）**: 针对高 TG（TG > 5.6 mmol/L 时首选）
- **处方级 Omega-3 脂肪酸（EPA/DHA）**: 显著降低 TG，对 AIP 改善明显
- **烟酸（尼克酸）**: 可升高 HDL-C，但需注意副作用（潮红、血糖影响）

**目标值**
- 首要目标：AIP < 0.24（理想 < 0.1）
- 次级目标：TG < 1.7 mmol/L（<150 mg/dL），HDL-C > 1.0 mmol/L（男）或 >1.3 mmol/L（女）

---

## 5. 特殊人群注意事项

- **糖尿病患者**: AIP 常显著升高（>0.35），需强化血糖控制与血脂管理
- **绝经后女性**: HDL-C 可能生理性下降，需结合心血管整体风险评估
- **长期素食者**: 若 AIP 异常，需排查精制碳水摄入过多导致的“碳水-高脂血症”

---

## 6. 监测频率

| 人群 | 监测频率 |
|-----|---------|
| 健康人群（AIP 正常） | 每年体检时复查 |
| AIP 临界升高（0.1-0.24） | 每 6 个月复查 |
| AIP 升高（>0.24）干预期 | 每 3 个月复查血脂 |
| 已启动药物治疗 | 4-6 周后复查，稳定后每 3-6 个月复查 |

> **总结**: AIP 是一个简便、低成本且预测价值高的心血管风险指标。其升高通常提示**胰岛素抵抗**和**致动脉粥样硬化脂质谱**，通过积极的饮食运动干预及必要的药物治疗，多数患者可在 3-6 个月内显著改善。
