# 脂蛋白分型（Lipoprotein Fractionation）详解

## 一、基本涵义

**脂蛋白分型**（又称**脂蛋白亚组分分析**）是一种通过物理或化学方法（如密度梯度超速离心、凝胶电泳、核磁共振波谱 NMR 等）将血浆中的脂蛋白按**密度、颗粒大小或电荷**进一步细分的检测技术。

与常规血脂四项（总胆固醇 TC、甘油三酯 TG、LDL-C、HDL-C）不同，分型检查能揭示脂蛋白的**“质”**而不仅是**“量”**，特别是能识别出致动脉粥样硬化性更强的小而密 LDL（sdLDL）颗粒，以及 HDL 的亚型分布，从而更精准地评估心血管疾病（CVD）风险。

### 主要分型类别
| 类别 | 缩写 | 主要功能 | 致动脉粥样硬化性 |
| :--- | :--- | :--- | :--- |
| **乳糜微粒** | CM | 运输外源性 TG | 极低（急性胰腺炎风险） |
| **极低密度脂蛋白** | VLDL | 运输内源性 TG | 中等 |
| **中间密度脂蛋白** | IDL | VLDL 代谢产物 | 中等偏高 |
| **低密度脂蛋白** | LDL | 运输胆固醇至外周 | **高**（尤其是 sdLDL 亚型） |
| **高密度脂蛋白** | HDL | 逆向转运胆固醇 | 保护性（HDL2 > HDL3） |
| **脂蛋白(a)** | Lp(a) | 遗传相关 | 独立强危险因素 |

---

## 二、常见异常类型及临床意义

### 1. LDL 颗粒大小/密度异常（最重要）
*   **Pattern B（B 型模式）**：以**小而密 LDL（sdLDL）**为主。颗粒小，易氧化，穿透动脉内膜能力强，致动脉粥样硬化性比大而轻 LDL（lbLDL）高 3-4 倍。
*   **Pattern A（A 型模式）**：以**大而轻 LDL**为主，相对无害。

### 2. HDL 亚型失衡
*   **HDL2 降低，HDL3 相对升高**：HDL2 是成熟的大颗粒，具有强抗氧化和胆固醇逆向转运能力；其降低提示心血管保护功能减弱。

### 3. 乳糜微粒（CM）血症
*   **血样外观**：血浆呈“奶油样”上层（4℃静置过夜）。
*   **电泳特征**：原点出现深染条带。

### 4. 宽β带（β-VLDL 升高）
*   电泳图谱中β区和前β区融合形成宽条带，提示 **III 型高脂蛋白血症（家族性异常β脂蛋白血症）**。

### 5. 脂蛋白(a) [Lp(a)] 升高
*   独立于 LDL 的遗传因素，升高即增加风险，不受饮食运动影响。

---

## 三、异常对应的可能原因

| 异常指标 | 主要代谢/病理原因 | 遗传/继发性因素 |
| :--- | :--- | :--- |
| **sdLDL 升高 (Pattern B)** | **胰岛素抵抗**、2 型糖尿病前期、高甘油三酯血症、肥胖（腹型）、高糖/高碳水饮食、久坐 | 家族性混合型高脂血症 |
| **HDL 亚型不良** | 缺乏运动、吸烟、胰岛素抵抗、肥胖、高 TG 血症 | CETP 基因多态性 |
| **CM 显著升高** | 急性酗酒、未控制的糖尿病、严重甲减 | **LPL 缺陷**（I 型）、**ApoC-II 缺陷** |
| **β-VLDL 升高 (Type III)** | 甲减、肾病综合征、糖尿病 | **ApoE2/E2 纯合子**（经典 III 型） |
| **VLDL 升高** | 过量饮酒、肥胖、高糖饮食、胰岛素抵抗 | 家族性高甘油三酯血症 |
| **Lp(a) 升高** | — | **LPA 基因kringle IV 重复序列多态性**（常染色体显性遗传） |

## 四、建议后续处理方案

### （一）生活方式干预（针对性调整）

#### 针对 **sdLDL 升高 / Pattern B**
*   **饮食**：采用**低碳水化合物**或**低升糖指数（低 GI）**饮食（减少精制碳水、添加糖），而非单纯低脂饮食。增加单不饱和脂肪酸（橄榄油、坚果）。
*   **减重**：减轻体重 5-10% 可显著改善 LDL 颗粒大小。
*   **运动**：每周 ≥150 分钟中等强度有氧运动（快走、游泳），可提高 LDL 颗粒直径，降低 sdLDL 比例。

#### 针对 **HDL 亚型不良**
*   **戒烟**：戒烟可迅速提升 HDL2 水平。
*   **运动**：耐力训练最有效提升 HDL2。

#### 针对 **高 TG / CM 血症**
*   **严格戒酒**：酒精是 VLDL 和 CM 合成的主要刺激物。
*   **极低脂饮食**：对于 CM 血症患者，需限制脂肪摄入 <20% 总热量。

### （二）药物干预（需医生评估）
*   **他汀类**：基础用药，主要降低 LDL 颗粒数量，对 sdLDL 有一定改善作用。
*   **贝特类**（非诺贝特等）：针对高 TG 和 sdLDL，改善 Pattern B。
*   **烟酸**：可升高 HDL2，降低 sdLDL，但需注意潮红和血糖影响。
*   **ω-3 脂肪酸**（高纯度 EPA/DHA）：针对严重高 TG（>5.6 mmol/L）。
*   **PCSK9 抑制剂**：强效降低 LDL 颗粒数，适用于高危患者。
*   **Apo(a) 反义寡核苷酸**（Pelacarsen）或 **siRNA**（Olpasiran）：针对 Lp(a) 极高患者（实验性/逐步上市）。

### （三）进一步详细检查

1.  **病因与代谢评估**：
    *   **糖代谢**：空腹血糖、胰岛素、C-肽、HbA1c、OGTT（排查胰岛素抵抗）。
    *   **甲状腺功能**：TSH、FT4（排查甲减）。
    *   **肾功能与尿蛋白**：排查肾病综合征。
    *   **载脂蛋白检测**：ApoB（颗粒数）、ApoA-I、ApoB/ApoA-I 比值。
    *   **炎症指标**：hs-CRP。

2.  **遗传学检查**（特定情况）：
    *   **ApoE 基因型**：若怀疑 III 型高脂蛋白血症（宽β带）。
    *   **LPL、ApoC-II、ApoA-V 基因测序**：若怀疑家族性 CM 血症。
    *   **LPA 基因型**：确认 Lp(a) 遗传风险。

3.  **靶器官损害评估**（评估是否已发生动脉粥样硬化）：
    *   **颈动脉超声**：IMT（内膜中层厚度）和斑块检测。
    *   **冠状动脉钙化评分（CAC）**：低剂量 CT，评估冠脉钙化负荷。
    *   **踝肱指数（ABI）**：排查外周动脉疾病。

## 五、适用人群建议

以下人群建议在常规血脂异常基础上，考虑进行脂蛋白分型：
1.  **早发冠心病家族史**（男性 <55 岁，女性 <65 岁发病）但常规 LDL-C 正常或轻度升高者。
2.  **代谢综合征/糖尿病患者**：即使 LDL-C 达标，仍需评估 sdLDL 残余风险。
3.  **反复胰腺炎发作**伴高 TG 者：明确 CM 和 VLDL 比例。
4.  **黄色瘤患者**：特别是掌纹黄色瘤（提示 III 型高脂血症）。
5.  **他汀治疗 LDL-C 已达标但仍发生心血管事件**（Residual Risk，残余风险）者。

> **注意**：脂蛋白分型目前多作为进阶风险评估工具，尚未完全纳入常规体检套餐，但对于中高危人群优化治疗策略具有重要价值。结果解读需结合临床由专科医生（心内科/内分泌科/血脂管理门诊）完成。
