**Heart & Lungs CT Scan（心肺CT扫描）** 是体检中用于评估胸腔内器官结构的高分辨率影像学检查。通常包括**胸部低剂量CT（LDCT，主要查肺）**和/或**冠状动脉CT血管造影（CTA，主要查心脏血管）**。

以下是该检查指标的详细解读：

---

### 一、 指标涵义与检查内容

| 检查类型 | 主要观察目标 | 体检常见应用 |
|---------|------------|------------|
| **胸部CT平扫**<br>（Low-dose CT） | 双肺实质、气管支气管、纵隔淋巴结、胸膜、胸壁、部分大血管（主动脉、肺动脉） | 肺癌筛查、肺炎、肺结节、肺气肿 |
| **冠状动脉钙化积分**<br>（CAC Score） | 冠状动脉壁的钙化斑块 | 心血管风险评估（Agatston积分） |
| **冠脉CTA**<br>（Coronary CTA） | 冠状动脉管腔狭窄程度、斑块性质 | 冠心病排查（需注射造影剂） |

**注意**：常规体检多指**胸部低剂量CT**+**钙化积分**，而非增强CTA。

---

### 二、 常见异常及可能原因

#### **A. 肺部异常**
| 异常表现 | 临床意义 | 可能原因 |
|---------|---------|---------|
| **肺结节**<br>（圆形阴影，<3cm） | 最常见发现，90%以上为良性 | • 炎性肉芽肿（陈旧结核、真菌感染）<br>• 良性肿瘤（错构瘤）<br>• 肺癌（磨玻璃结节需警惕）<br>• 转移瘤（有肿瘤史者） |
| **磨玻璃影（GGO）** | 肺泡部分填充或间质增厚 | • 炎症/感染（可吸收）<br>• 肺泡出血<br>• 不典型腺瘤样增生（AAH）<br>• 原位腺癌（AIS）/微浸润腺癌（MIA） |
| **肺气肿/肺大疱** | 肺泡壁破坏，气体滞留 | • 长期吸烟（COPD）<br>• 老年性改变<br>• α-1抗胰蛋白酶缺乏 |
| **肺纤维化/网格影** | 肺间质增厚、瘢痕化 | • 特发性肺纤维化<br>• 结缔组织病（类风湿、硬皮病）<br>• 尘肺（职业暴露）<br>• 药物性肺损伤 |
| **肺炎/实变** | 肺组织密度增高 | • 细菌/病毒/真菌感染<br>• 吸入性肺炎 |
| **胸膜增厚/积液** | 胸膜病变 | • 陈旧结核、石棉暴露<br>• 心衰（积液）、恶性肿瘤胸膜转移 |

#### **B. 心脏及大血管异常**
| 异常表现 | 临床意义 | 可能原因 |
|---------|---------|---------|
| **冠脉钙化**<br>（CAC Score >0） | 冠状动脉粥样硬化标志 | • 年龄增长（生理性）<br>• 高血压、高血脂、糖尿病<br>• 吸烟、早发心血管病家族史 |
| **冠脉狭窄**<br>（CTA发现） | 心肌缺血风险 | • 动脉粥样硬化斑块<br>• 血管痉挛（少见） |
| **心肌肥厚** | 心脏负荷增加 | • 高血压（长期未控制）<br>• 肥厚型心肌病<br>• 主动脉瓣狭窄 |
| **心包积液** | 心包腔液体积聚 | • 病毒感染<br>• 自身免疫病<br>• 肿瘤、甲减 |
| **主动脉硬化/扩张** | 血管衰老/病变 | • 动脉粥样硬化<br>• 马凡综合征（扩张）<br>• 高血压 |

### 三、 异常对应的后续处理建议

#### **1. 肺结节管理策略**（基于Fleischner Society指南）

| 结节类型与大小 | 风险分级 | 建议处理 |
|--------------|---------|---------|
| **实性结节 <6mm** | 低危 | 通常无需随访（吸烟者可12个月后复查） |
| **实性结节 6-8mm** | 中危 | 6-12个月复查CT，若无变化18-24个月再查 |
| **实性结节 >8mm** | 高危 | • 3个月复查CT<br>• 或PET-CT检查（SUV值判断代谢活性）<br>• 必要时CT引导下穿刺活检或手术切除 |
| **纯磨玻璃结节（pGGN）<6mm** | - | 通常无需随访，或可选2-4年复查 |
| **纯磨玻璃结节 ≥6mm** | - | 6-12个月复查，持续存在则每2年复查至5年（警惕惰性腺癌） |
| **混合磨玻璃结节** | - | 3-6个月复查，若实性成分≥6mm或增长，建议切除 |

#### **2. 冠脉钙化积分（CAC Score）管理**

| 积分值 | 心血管风险 | 建议处理 |
|-------|-----------|---------|
| **0** | 极低风险 | 维持健康生活方式，5-10年重复检测 |
| **1-99** | 轻度升高 | 强化生活方式干预，考虑他汀类药物治疗（尤其>40岁） |
| **100-399** | 中度升高 | 启动他汀治疗，控制LDL-C<1.8mmol/L，建议负荷心电图或心肌核素检查 |
| **≥400** | 高危 | 强烈建议冠脉造影或冠脉CTA（如未做），必要时支架评估，严格控制三高 |

#### **3. 生活方式调整（通用建议）**

**针对肺部：**
- **立即戒烟**：吸烟是肺癌和COPD的首要危险因素，戒烟后10年肺癌风险可降低50%。
- **避免暴露**：远离二手烟、厨房油烟、职业粉尘（石棉、硅尘）、空气污染（PM2.5高时戴口罩）。
- **呼吸锻炼**：腹式呼吸、缩唇呼吸，增强肺功能。
- **疫苗接种**：每年流感疫苗、肺炎链球菌疫苗（>65岁或COPD患者）。

**针对心脏：**
- **饮食**：采用地中海饮食或DASH饮食，限盐（<5g/天）、限饱和脂肪，增加膳食纤维和Omega-3。
- **运动**：每周≥150分钟中等强度有氧运动（快走、游泳），增强心肺功能。
- **控制体重**：BMI控制在18.5-23.9，腰围（男<90cm，女<85cm）。
- **压力管理**：保证睡眠，避免长期精神紧张。

#### **4. 需进一步专科检查的情况**

| 异常发现 | 建议检查 | 就诊科室 |
|---------|---------|---------|
| 可疑恶性结节 | PET-CT、支气管镜活检、经皮肺穿刺、胸腔镜手术 | 胸外科/呼吸科 |
| 间质性肺病改变 | 高分辨率CT（HRCT）、肺功能检测、自身抗体谱 | 呼吸科/风湿免疫科 |
| 冠脉中重度狭窄 | 冠脉造影（CAG）、心肌灌注显像、心脏彩超 | 心内科 |
| 主动脉扩张>4cm | 主动脉全程CTA、心脏彩超 | 心外科/血管外科 |

---

### 四、 重要注意事项

1. **辐射剂量**：低剂量胸部CT辐射约1-2mSv（相当于半年自然本底辐射），相对安全；但短期内（<6个月）避免重复做增强CT。
2. **假阳性问题**：约20-30%体检者会发现肺结节，但绝大多数为良性，避免过度焦虑。
3. **造影剂风险**：如需做心脏CTA，需评估肾功能（eGFR）和碘过敏史，糖尿病患者需停用二甲双胍48小时。
4. **年龄因素**：<40岁无吸烟史者，一般不建议常规做肺癌筛查CT；>50岁吸烟者强烈建议每年低剂量CT。

**建议**：拿到报告后应携带影像资料（DICOM文件或胶片）就诊**呼吸科**或**心内科**，由医生结合您的**吸烟史、家族史、肿瘤标志物、血脂水平**进行综合判断，切勿仅凭报告单自行对号入座。
