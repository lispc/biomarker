**尿肾小管上皮细胞（Urine Renal Tubular Epithelial Cells, RTECs）** 是尿液沉渣镜检中的一个重要指标，直接反映**肾实质（特别是肾小管）**的损伤情况。

---

### 一、指标涵义与正常参考

| 特征 | 说明 |
|------|------|
| **细胞来源** | 来源于肾小管（近曲小管、远曲小管、集合管）的脱落细胞 |
| **形态特点** | 圆形或多边形，体积比白细胞大（约15-25μm），胞核大而圆，胞质丰富，可能含有颗粒或空泡 |
| **正常值** | **健康人尿液中应为阴性或极少量**（0-偶见/HP）。由于肾小管细胞更新缓慢且黏附紧密，正常尿液中几乎不应出现 |
| **临床意义** | **特异性高**：一旦出现，通常提示肾小管结构受损（区别于来自尿道/阴道的鳞状上皮细胞） |

---

### 二、常见异常情况及对应病因

#### 1. **轻度增多（少量散在）**
- **可能原因**：
  - 剧烈运动后的一过性损伤
  - 轻度脱水或发热导致的肾灌注不足
  - 药物影响（如短期使用NSAIDs、某些抗生素）
  - 早期糖尿病肾病或高血压肾损害

#### 2. **显著增多（成堆出现或伴有管型）**
- **高度提示急性肾小管损伤**：
  - **急性肾小管坏死（ATN）**：缺血性（休克、严重脱水）或肾毒性（氨基糖苷类抗生素、造影剂、重金属）
  - **急性间质性肾炎（AIN）**：药物过敏（青霉素类、质子泵抑制剂、NSAIDs）或感染
  - **肾小球疾病**：急进性肾小球肾炎、狼疮性肾炎活动期
  - **肾移植排斥反应**：急性细胞性排斥反应的早期标志
  - **病毒感染**：巨细胞病毒（CMV）、BK多瘤病毒肾病

#### 3. **伴有特殊形态**
- **含脂肪滴**：肾病综合征（肾小管上皮细胞吸收尿中脂蛋白形成）
- **含铁血黄素颗粒**：陈旧性肾实质出血（如溶血或肌肉损伤后）

---

### 三、建议后续处理

#### 🔍 **第一步：核实与鉴别（排除假阳性）**
- **复查尿常规**：排除标本污染（女性需避开月经期，留取清洁中段尿）
- **尿沉渣镜检**：由检验科医师确认是否为真正的肾小管上皮细胞（需与移行上皮细胞、深层鳞状上皮细胞鉴别）
- **尿沉渣染色**：如 Sternheimer-Malbin 染色或 Wright 染色辅助识别

#### 🩺 **第二步：医学评估与检查**
| 检查项目 | 目的 |
|---------|------|
| **血液检查** | 血肌酐、尿素氮、胱抑素C（评估GFR）、电解质、血气分析 |
| **肾小管功能专项** | 尿β2-微球蛋白、尿α1-微球蛋白、NAG酶、NGAL、KIM-1（敏感反映肾小管损伤） |
| **尿液分析** | 尿蛋白定量（区分肾小球性与肾小管性蛋白尿）、尿电解质、尿渗透压 |
| **影像学** | 肾脏超声（看大小、皮质厚度、梗阻）、必要时CT/MRI |
| **免疫学检查** | 抗核抗体（ANA）、ANCA、抗GBM抗体（排除自身免疫病） |
| **病毒学** | BK病毒DNA、CMV-PP65（移植患者必查） |
| **肾活检** | 如病因不明或肾功能快速恶化，需病理确诊 |

#### 🏃 **第三步：生活方式与医学管理**

**立即调整：**
- **停用肾毒性药物**：在医生指导下暂停NSAIDs（布洛芬、双氯芬酸）、含马兜铃酸中药、氨基糖苷类抗生素、造影剂
- **充分水化**：每日饮水2000-2500ml（心功能允许情况下），避免肾前性缺血
- **控制基础病**：严格控糖（HbA1c<7%）、降压（目标<130/80 mmHg，优选ACEI/ARB类但需监测肌酐）
- **避免肾毒性暴露**：重金属（铅、汞、镉）、有机溶剂、不明成分草药

**专科随访：**
- **肾内科就诊**：尤其当合并血尿、蛋白尿或肾功能异常时
- **定期监测**：每1-2周复查尿常规，每月复查肾功能直至正常
- **移植患者**：调整免疫抑制剂方案（如降低他克莫司血药浓度，使用西多福韦治疗BK病毒）

---

### 四、特别注意事项

1. **与鳞状上皮细胞区分**：鳞状上皮细胞大而扁平，来自尿道远端或阴道，临床意义小；RTECs圆而小，提示肾实质问题。
2. **动态观察**：单次少量RTECs可能无意义，但**持续存在或进行性增加**必须重视。
3. **伴随症状警示**：如出现少尿、水肿、恶心、腰痛，提示急性肾损伤（AKI），需急诊处理。

**总结**：尿肾小管上皮细胞是**肾小管损伤的"哨兵指标"**。发现异常后，关键在于**寻找并去除损伤因素**（药物、毒素、缺血），同时评估肾功能储备，防止进展为慢性肾病（CKD）。
