**Sodium（钠，血清钠）**是血液电解质检查中最基础也是最关键的指标之一，主要反映体内水-电解质平衡状态。以下从临床角度为您系统梳理：

---

### 一、基本涵义与参考范围

| 项目 | 说明 |
|------|------|
| **生理作用** | 维持细胞外液渗透压、调节体液容量、保障神经肌肉兴奋性和动作电位传导 |
| **检测形式** | 血清钠（Serum Na⁺），通常包含在"电解质四项/七项"或"生化全项"中 |
| **正常参考值** | **135-145 mmol/L**（不同实验室可能略有差异，通常为136-146 mmol/L） |
| **关键概念** | 血钠异常通常反映的是**水代谢紊乱**（相对或绝对的水过剩/不足），而非单纯的钠摄入问题 |

---

### 二、常见异常类型及临床意义

#### 🔻 **低钠血症（Hyponatremia）—— 更常见**
**定义**：血清钠 < 135 mmol/L  
**分级**：
- 轻度：130-135 mmol/L（通常无症状）
- 中度：125-129 mmol/L（可能出现恶心、头痛）
- 重度：< 125 mmol/L（嗜睡、抽搐、昏迷，需紧急处理）

**常见原因分类**：

| 类型 | 机制 | 典型病因 |
|------|------|----------|
| **稀释性低钠**<br>（体内水过多） | 水排泄障碍或摄入过多 | • 抗利尿激素分泌异常综合征（SIADH）<br>• 心力衰竭、肝硬化腹水<br>• 甲状腺功能减退<br>• 精神性多饮<br>• 术后应激反应 |
| **消耗性低钠**<br>（钠丢失过多） | 经肾或肾外途径失钠 | • 利尿剂（噻嗪类最常见）<br>• 呕吐、腹泻、胃肠减压<br>• 肾上腺皮质功能减退（Addison病）<br>• 脑盐耗综合征（CSWS） |
| **假性低钠** | 实验室干扰 | • 严重高脂血症或高蛋白血症（血浆固态成分增多，水相减少）<br>• 血糖极高（需校正：血糖每升高100mg/dL，血钠下降约1.6 mmol/L） |

#### 🔺 **高钠血症（Hypernatremia）—— 相对少见但危险**
**定义**：血清钠 > 145 mmol/L  
**分级**：
- 轻度：146-150 mmol/L
- 中度：151-157 mmol/L
- 重度：> 157 mmol/L（意识障碍、颅内出血风险）

**常见原因**：

| 类型 | 机制 | 典型病因 |
|------|------|----------|
| **水缺乏型**<br>（最常见） | 纯水丢失或摄入不足 | • 尿崩症（中枢性或肾性）<br>• 高热、大量出汗、烧伤<br>• 意识障碍患者无法饮水<br>• 渗透性利尿（未控制的高血糖） |
| **钠过载型** | 钠摄入过多或排出减少 | • 过量输注高渗盐水/碳酸氢钠<br>• 原发性醛固酮增多症<br>• Cushing综合征 |

---

### 三、发现异常后的处理建议

#### 🚨 **紧急评估（需立即就医的情况）**
- 血钠 < 120 mmol/L 或 > 155 mmol/L
- 伴随意识改变、癫痫发作、严重呕吐
- 急性发生（< 48小时）的低钠/高钠（脑水肿或脱髓鞘风险高）

#### 🔍 **进一步检查建议**

| 检查项目 | 目的 |
|----------|------|
| **尿钠测定** | 鉴别肾性失钠（尿钠>20 mmol/L）vs 肾外失钠（尿钠<10 mmol/L） |
| **血尿渗透压** | 评估SIADH（尿渗透压不适当地升高）或尿崩症（尿渗透压降低） |
| **甲状腺功能、皮质醇** | 排除甲减、肾上腺功能不全 |
| **BNP/心脏超声** | 评估心衰导致的稀释性低钠 |
| **头颅CT/MRI** | 怀疑脑肿瘤、蛛网膜下腔出血（引起SIADH）或脑性盐耗 |
| **血糖、血脂** | 排除假性低钠 |

#### 🛠️ **生活方式与干预措施**

**对于轻度低钠血症（130-135 mmol/L）**：
- **限制液体摄入**：若确诊为SIADH或稀释性低钠，每日饮水控制在800-1200ml（需医生指导）
- **调整饮食**：避免低盐饮食（除非医生特别要求），适量增加咸味食物；但**严禁快速大量补盐**
- **复查时机**：1-2周后复查，避免剧烈运动后立即检测

**对于轻度高钠血症**：
- **补充水分**：少量多次饮水，目标每日尿量>1.5L
- **监测出入量**：记录每日饮水量和尿量
- **避免高盐食物**：减少腌制食品、加工肉类摄入

**药物相关**：
- 如正在服用利尿剂、抗抑郁药（SSRI）、抗癫痫药（卡马西平），需告知医生评估是否与低钠相关，**切勿自行停药**

#### ⚠️ **重要医学警示**
- **低钠血症纠正速度**：慢性低钠血症（>48小时）纠正速度不宜过快（通常<8-10 mmol/L/24小时），否则可能诱发**渗透性脱髓鞘综合征**（ODS），导致永久性神经损伤。
- **假性异常**：若血钠结果与临床表现不符（如无症状但血钠极低），需复查并排除高脂血症干扰。

---

### 四、总结流程图

```
体检发现血钠异常
    ↓
确认是否急性症状（意识障碍、抽搐）
    ↓
有 → 急诊处理
无 → 复查电解质（排除实验室误差）
    ↓
结合尿钠、渗透压、临床表现判断类型
    ↓
针对性检查（激素、影像）→ 病因治疗
```

**建议**：单次轻度异常（如134 mmol/L）且无症状时，可在调整饮食后1-2周复查；若持续异常或伴有任何不适症状，应尽早就诊内分泌科或肾内科。

*注：以上信息仅供科普参考，具体诊疗需结合临床由医生判断。*
