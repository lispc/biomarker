**Estimated Average Glucose（eAG，估算平均血糖）** 是一项通过糖化血红蛋白（HbA1c）换算而来的指标，用于反映过去 **2-3 个月内** 的平均血糖水平。与即时血糖（空腹或餐后）不同，eAG 代表的是长期的血糖控制状况。

---

### 一、基本涵义与换算

**核心概念：**
- **计算基础**：基于 HbA1c（糖化血红蛋白）的百分比值推算
- **标准公式**：`eAG (mg/dL) = 28.7 × HbA1c (%) - 46.7`
- **单位换算**：1 mmol/L = 18 mg/dL（中国常用 mmol/L，美国常用 mg/dL）
- **时间跨度**：反映红细胞生命周期（约 8-12 周）内的平均血糖暴露

**参考范围对照表：**

| HbA1c | eAG (mg/dL) | eAG (mmol/L) | 临床解读 |
|-------|-------------|--------------|----------|
| < 5.7% | < 117 | < 6.5 | 正常 |
| 5.7-6.4% | 117-137 | 6.5-7.6 | 糖尿病前期（空腹血糖受损） |
| ≥ 6.5% | ≥ 140 | ≥ 7.8 | 糖尿病 |
| ≥ 7.0% | ≥ 154 | ≥ 8.6 | 糖尿病控制目标（一般成人） |

---

### 二、常见异常及可能原因

#### 🔴 **异常升高（最常见）**
**表现**：eAG > 140 mg/dL（7.8 mmol/L），对应 HbA1c ≥ 6.5%

**可能原因：**
1. **糖尿病或糖尿病前期**：胰岛素抵抗或分泌不足
2. **未控制的已知糖尿病**：治疗方案不当或依从性差
3. **应激性高血糖**：急性感染、创伤、手术、心肌梗死
4. **药物影响**：糖皮质激素、噻嗪类利尿剂、部分抗精神病药物
5. **特殊生理状态**：妊娠（妊娠糖尿病）、库欣综合征、肢端肥大症
6. **检验干扰**：缺铁性贫血（可能假性升高 HbA1c，但 eAG 计算值会偏高）

#### 🟡 **异常偏低（较少见）**
**表现**：eAG < 70 mg/dL（3.9 mmol/L）

**可能原因：**
1. **低血糖倾向**：胰岛素瘤、肾上腺皮质功能减退
2. **药物过量**：降糖药（胰岛素、磺脲类）过量
3. **检验干扰**：血红蛋白变异体（如 HbS、HbC）、溶血性贫血（缩短红细胞寿命，导致 HbA1c 假性降低）
4. **严重肝肾功能不全**：影响糖代谢

### 三、异常对应的后续处理建议

#### **情况 A：eAG 升高（糖尿病前期/糖尿病）**

**立即行动：**
1. **复查确认**：1-2 周内复查空腹血糖 + HbA1c，必要时做 **OGTT（口服葡萄糖耐量试验）**
2. **详细评估**：
   - 空腹胰岛素 + C 肽（评估胰岛功能）
   - 尿微量白蛋白/肌酐比（早期肾损害筛查）
   - 眼底检查、神经传导速度（并发症筛查）
   - 血脂、肝功能、肾功能

**生活方式干预（基石治疗）：**
- **饮食**：采用低升糖指数（GI）饮食，控制碳水化合物总量（占总热量 45-60%），增加膳食纤维（>25g/天），限制精制糖和饱和脂肪
- **运动**：每周 ≥150 分钟中等强度有氧运动（如快走、游泳）+ 每周 2 次抗阻训练
- **体重管理**：超重人群减重 5-7% 可显著改善胰岛素敏感性
- **监测**：开始家庭血糖监测（空腹 + 餐后 2 小时）

**医疗干预：**
- 若 eAG ≥ 154 mg/dL（8.6 mmol/L）或合并心血管疾病，需内分泌科就诊，考虑启动二甲双胍等药物治疗
- 妊娠女性需立即产科-内分泌联合管理

#### **情况 B：eAG 偏低**

**排查步骤：**
1. **排除检验误差**：复查血常规（看是否有贫血）、血红蛋白电泳（排除血红蛋白病）
2. **症状评估**：询问是否有心悸、出汗、饥饿感等低血糖症状
3. **72 小时动态血糖监测（CGM）**：确认实际血糖波动情况
4. **病因检查**：胰岛素抗体、皮质醇节律、腹部 CT（排查胰岛素瘤）

**处理：**
- 如为药物过量：调整降糖方案
- 如为胰岛素瘤：外科会诊

---

### 四、重要注意事项

1. **eAG 是"估算"值**：基于 HbA1c 推算，不能替代日常血糖监测。某些人群（如贫血、血红蛋白病患者）eAG 可能不准确。
2. **个体化目标**：
   - 年轻、病程短、无并发症：eAG 目标 < 140 mg/dL（HbA1c < 6.5%）
   - 老年人、有低血糖风险：eAG 目标可放宽至 < 169 mg/dL（HbA1c < 7.5%）
3. **急性血糖变化滞后性**：eAG 无法反映近期（2 周内）血糖的急剧变化。

**建议**：首次发现 eAG 异常，应携带报告至内分泌科就诊，制定个体化管理方案，切勿自行诊断或调整药物。
