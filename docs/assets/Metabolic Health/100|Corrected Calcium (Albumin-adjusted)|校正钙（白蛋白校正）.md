**校正钙（Albumin-adjusted Calcium）**是临床评估钙代谢状态的重要指标。由于血液中约40-50%的钙与白蛋白结合，当血清白蛋白水平异常（如肝病、肾病综合征）时，总钙水平会"假性"升高或降低，而校正钙能更真实反映具有生理活性的游离钙水平。

---

## 一、核心概念与计算公式

### 为什么需要校正？
- **总钙**包含结合钙（与白蛋白结合）和游离钙（离子钙，具生理活性）
- 白蛋白降低 → 总钙降低（但游离钙可能正常）
- 白蛋白升高 → 总钙升高（但游离钙可能正常）

### 常用校正公式
- **传统公式**：`校正钙(mg/dL) = 血清总钙 + 0.8 × (4.0 - 白蛋白g/dL)`
- **国际单位**：`校正钙(mmol/L) = 血清总钙 + 0.02 × (40 - 白蛋白g/L)`

**参考范围**：2.2–2.6 mmol/L（8.8–10.4 mg/dL），不同实验室略有差异。

## 二、常见异常及可能原因

### 🔴 校正钙偏高（高钙血症，>2.6 mmol/L）

| 常见原因 | 机制/特点 |
|---------|----------|
| **原发性甲状旁腺功能亢进** | 最常见原因，PTH自主分泌过多，常伴血磷降低 |
| **恶性肿瘤** | 骨转移破坏、分泌甲状旁腺激素相关蛋白(PTHrP) |
| **维生素D中毒** | 过量补充导致肠道钙吸收增加 |
| **甲状腺功能亢进** | 骨代谢加速，钙释放增加 |
| **噻嗪类利尿剂** | 减少尿钙排泄 |
| **肉芽肿性疾病** | 结节病、结核等，巨噬细胞将25-OH-D转化为活性1,25-OH-D |
| **长期卧床/制动** | 骨质疏松，骨钙流失 |
| **乳碱综合征** | 过量摄入钙剂+可吸收碱剂 |

### 🔵 校正钙偏低（低钙血症，<2.2 mmol/L）

| 常见原因 | 机制/特点 |
|---------|----------|
| **甲状旁腺功能减退** | 甲状腺/甲状旁腺术后、自身免疫性疾病 |
| **维生素D缺乏** | 营养不良、吸收不良综合征、日照不足 |
| **慢性肾病(CKD)** | 磷潴留、活性维生素D合成障碍、继发性甲旁亢晚期 |
| **低镁血症** | 严重缺镁抑制PTH分泌和作用（需先补镁）|
| **急性胰腺炎** | 脂肪酸与钙结合沉积 |
| **高磷血症** | 肾衰、横纹肌溶解，钙磷乘积升高致钙沉积 |
| **药物** | 双膦酸盐、降钙素、苯妥英钠、西那卡塞 |

## 三、后续处理建议

### 1️⃣ 确认检测准确性
- **复查**：排除溶血、止血带使用时间过长等实验误差
- **检测离子钙**：如条件允许，直接检测血清游离钙（Ionized Calcium）是最准确的
- **注意pH影响**：呼吸性碱中毒时游离钙降低（出现手足麻木），酸中毒时相反

### 2️⃣ 针对性进一步检查

#### 若校正钙升高：
- **必查**：甲状旁腺激素(PTH)、血磷、血镁、肾功能
- **维生素D代谢**：25-羟维生素D、1,25-二羟维生素D
- **肿瘤筛查**：根据年龄和症状选择（骨扫描、肿瘤标志物、胸部CT等）
- **尿钙测定**：鉴别家族性低尿钙高钙血症(FHH) vs 原发性甲旁亢
- **骨密度检查**：长期高钙需评估骨质疏松风险

#### 若校正钙降低：
- **必查**：PTH（鉴别甲状旁腺功能状态）、血磷、血镁
- **维生素D水平**：25-羟维生素D（反映营养状态）
- **肾功能+电解质**：评估CKD-MBD
- **肝功能**：确认白蛋白水平（排除校正误差）
- **淀粉酶/脂肪酶**：排除急性胰腺炎
- **骨代谢标志物**：如怀疑骨软化症

### 3️⃣ 生活方式调整

#### 高钙血症：
- **充分水化**：每日饮水2-3升，预防肾结石
- **限制钙摄入**：暂停钙片、减少高钙食物（如短期内>1000mg/天）
- **避免维生素D补充**：停用复合维生素及维生素D制剂
- **适度活动**：避免长期卧床导致骨钙流失
- **限盐**：高钠饮食增加尿钙排泄，但高钙时需谨慎（遵医嘱）

#### 低钙血症：
- **增加膳食钙**：奶制品、深绿色蔬菜、豆腐、芝麻
- **维生素D补充**：日晒（每日15-30分钟）或补充剂（需检测后遵医嘱）
- **避免高磷食物**：加工食品、碳酸饮料（尤其CKD患者）
- **监测症状**：注意手足麻木、肌肉痉挛、抽搐等

### 4️⃣ 医疗干预指征

**需急诊处理**（高钙危象）：
- 血钙>3.5 mmol/L，或出现意识模糊、严重心律失常、顽固性呕吐、脱水

**需急诊处理**（低钙危象）：
- 手足搐搦、喉痉挛、癫痫发作、QT间期延长致心律失常

**专科转诊**：
- 内分泌科：疑似甲旁亢、复杂钙磷代谢紊乱
- 肾内科：CKD-MBD
- 肿瘤科：疑似恶性肿瘤相关高钙

## 四、特别提醒

1. **校正钙的局限性**：当白蛋白<20g/L或>50g/L时，校正公式误差较大，建议直接检测离子钙。
2. **假性异常**：严重高脂血症、多发性骨髓瘤（异常蛋白结合钙）时，校正钙可能不准确。
3. **慢性变化**：长期轻度低钙（如术后甲旁减）需终身补钙和维生素D监测，避免肾结石和异位钙化。

建议拿到异常结果后，**切勿自行大量补钙或限钙**，应在1-2周内复查并咨询内分泌专科医生，明确病因后再制定治疗方案。
