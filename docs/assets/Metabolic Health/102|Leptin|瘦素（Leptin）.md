**Leptin（瘦素）** 是一种由白色脂肪组织（White Adipose Tissue）分泌的多肽激素，主要由肥胖基因（*Ob* gene）编码。它是机体能量代谢平衡的核心调控因子之一，通过作用于下丘脑的特异性受体（LEPR），抑制食欲、增加能量消耗，并参与调节生殖、免疫、骨代谢等生理功能。

需要注意的是，**瘦素目前并非临床常规体检项目**，通常仅在特定代谢疾病研究、难治性肥胖评估、或怀疑先天性瘦素缺乏/脂肪代谢异常时进行检测。

---

### 一、指标解读参考范围
- **正常范围**（空腹血清）：因性别和体脂率差异较大
  - 男性：约 0.5–13.8 ng/mL（或 0.5–13.8 μg/L）
  - 女性：约 1.2–27.0 ng/mL（略高于男性，与较高体脂相关）
- **单位换算**：1 ng/mL = 1 μg/L

---

### 二、异常情况及临床意义

#### 1. 瘦素水平升高（高瘦素血症，Hyperleptinemia）
**最常见**，通常与体脂量增加成正比，但存在"瘦素抵抗"现象。

| 可能原因 | 详细说明 |
|---------|---------|
| **肥胖/超重** | 脂肪组织增加导致分泌代偿性升高，但下丘脑对瘦素敏感性下降（瘦素抵抗） |
| **代谢综合征** | 常伴随胰岛素抵抗、血脂异常，形成"代谢炎症"循环 |
| **炎症状态** | 瘦素本身是促炎细胞因子，感染、自身免疫病、慢性炎症可致升高 |
| **甲状腺功能减退** | 代谢率降低伴随体重增加，瘦素水平通常升高 |
| **多囊卵巢综合征（PCOS）** | 常伴随肥胖和胰岛素抵抗 |
| **肾功能不全** | 瘦素主要经肾脏清除，eGFR下降时清除减少 |
| **糖皮质激素使用** | 促进脂肪重分布和瘦素分泌 |

#### 2. 瘦素水平降低（低瘦素血症，Hypoleptinemia）
相对少见，但临床意义重要。

| 可能原因 | 详细说明 |
|---------|---------|
| **先天性瘦素缺乏** | *LEP* 基因突变，极罕见，表现为幼年起病严重肥胖、食欲亢进、性腺功能低下 |
| **脂肪萎缩症** | 全身性或局部性脂肪组织缺失（如HIV相关脂肪萎缩、先天性全身性脂肪营养不良） |
| **严重营养不良/神经性厌食** | 体脂率极低时分泌显著下降 |
| **慢性消耗性疾病** | 晚期恶性肿瘤、重度心衰、慢性阻塞性肺病恶病质期 |
| **高强度耐力运动** | 长期能量负平衡导致暂时性降低 |

### 三、异常后的建议处理方案

#### 针对瘦素升高（多数情况）

**生活方式干预（一线方案）：**
- **减重目标**：即使减重5-10%，也能显著改善瘦素敏感性
  - 饮食：采用地中海饮食或低碳水化合物饮食，减少果糖和精制碳水摄入（果糖直接促进肝脏脂肪合成并干扰瘦素信号）
  - 运动：每周≥150分钟中等强度有氧运动（如快走、游泳）结合抗阻训练，运动可改善下丘脑瘦素受体敏感性
  - 睡眠：保证7-9小时睡眠，睡眠剥夺会加剧瘦素抵抗
- **抗炎饮食**：增加Omega-3（深海鱼、亚麻籽）、多酚类（绿茶、浆果）摄入，减少反式脂肪

**医学评估（建议检查）：**
- 代谢指标：空腹血糖、胰岛素、HOMA-IR指数、HbA1c、血脂谱
- 炎症指标：hs-CRP、IL-6（瘦素与炎症正相关）
- 内分泌评估：甲状腺功能（TSH、FT4）、皮质醇节律、性激素（女性查睾酮、LH/FSH排除PCOS）
- 肾功能：eGFR、尿蛋白（排除肾性清除减少）
- **注**：如BMI>35且伴有并发症，可咨询内分泌科评估代谢手术（减重手术可显著降低瘦素水平并改善敏感性）

#### 针对瘦素降低（罕见但需重视）

**紧急评估（如显著降低）：**
- 如伴随严重肥胖：需基因检测排除先天性瘦素缺乏（*LEP* 或 *LEPR* 基因突变）
- 如伴随脂肪缺失：排查脂肪营养不良综合征（查血清甘油三酯、肝酶、胰腺影像排除脂肪萎缩性糖尿病）

**针对性治疗：**
- **先天性瘦素缺乏**：可使用重组甲硫氨酰瘦素（Metreleptin）替代治疗，可显著降低体重、改善代谢和性腺功能
- **获得性脂肪萎缩**：Metreleptin也被批准用于此适应症，需专科医生评估
- **营养不良**：营养康复计划，逐步增加热量摄入，避免再喂养综合征

---

### 四、特别提醒
1. **不要试图"补充瘦素"减肥**：对于普通肥胖者，外源性瘦素治疗效果极差（因存在瘦素抵抗），且可能引发中和抗体和代谢副作用。
2. **检测标准化**：瘦素水平受昼夜节律（夜间较高）、月经周期（黄体期较高）和进食状态影响，建议空腹、上午8-10点采血。
3. **综合判断**：单一瘦素数值临床意义有限，必须结合体脂率、腰围、炎症指标和代谢参数综合评估。

如体检发现瘦素异常，建议携带报告至**内分泌代谢科**就诊，由医生判断是否需要进一步行口服糖耐量试验（OGTT）、体脂分析或基因筛查。
