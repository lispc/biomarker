**促甲状腺激素（Thyroid-Stimulating Hormone, TSH）** 是评估甲状腺功能最敏感、最核心的指标。以下从生理机制、异常解读到应对策略进行系统说明：

---

### 一、基本涵义与生理机制

**定义**：TSH 由脑垂体（垂体前叶）分泌，作用是"命令"甲状腺合成和释放甲状腺激素（T3、T4）。

**核心逻辑**（负反馈调节）：
- 当血液中甲状腺激素（T3/T4）**不足**时 → 垂体分泌更多 TSH（升高）→ 刺激甲状腺加班工作
- 当血液中甲状腺激素（T3/T4）**过多**时 → 垂体抑制 TSH 分泌（降低）→ 让甲状腺休息

**正常参考范围**：通常为 **0.4–4.0 mIU/L**（不同实验室可能略有差异，部分采用 0.27–4.2）。  
**特殊人群**：孕妇（尤其孕早期）上限通常需控制在 2.5–3.0 mIU/L 以下；老年人（>70岁）TSH 轻度升高（如 5–7 mIU/L）可能为正常衰老现象。

---

### 二、TSH 升高的常见情况（提示甲状腺功能减退倾向）

#### 1. 解读逻辑
TSH 升高意味着垂体在"拼命催促"甲状腺工作，通常提示甲状腺激素不足（甲减）。

#### 2. 可能原因
| 类型 | 具体原因 | 特点 |
|------|---------|------|
| **原发性甲减**（最常见） | 桥本甲状腺炎（慢性淋巴细胞性甲状腺炎） | 自身免疫攻击，常伴 TPOAb/TgAb 抗体阳性 |
| | 甲状腺切除术后或碘-131治疗后 | 甲状腺组织减少 |
| | 碘缺乏或碘过量 | 缺碘地区或长期大量摄入海带/紫菜/含碘药物 |
| | 药物影响 | 锂剂、胺碘酮、干扰素、部分抗癫痫药 |
| **亚临床甲减** | TSH 升高（通常 4.5–10），但 FT3/FT4 正常 | 早期或轻度功能不足，可能无症状 |
| **垂体 TSH 瘤**（罕见） | 垂体肿瘤自主分泌 TSH | TSH 显著升高且 FT3/FT4 也升高（与原发性甲减相反） |
| **恢复期** | 亚急性甲状腺炎恢复期 | 暂时性升高，随后可能恢复正常 |

#### 3. 后续处理建议
- **复查确认**：1–3 个月后复查 TSH + 游离甲状腺激素（FT3、FT4），排除实验室误差或暂时性波动。
- **抗体检测**：查 **TPOAb**（甲状腺过氧化物酶抗体）和 **TgAb**（甲状腺球蛋白抗体）排查桥本甲状腺炎。
- **影像学检查**：甲状腺超声（看是否有萎缩、弥漫性病变或结节）。
- **药物治疗**：
  - 若 TSH > 10 mIU/L，或 TSH 4.5–10 且伴有症状（疲劳、怕冷、体重增加、血脂异常）或备孕/怀孕，通常需口服 **左甲状腺素钠（优甲乐）** 替代治疗。
  - 若 TSH 轻度升高（4.5–10）且无症状，可暂观察，每 6–12 个月复查。
- **生活方式**：
  - 适量补碘（但桥本患者避免过量碘摄入，如海带、紫菜）。
  - 保证硒、锌、维生素 D 摄入（坚果、鱼类、蛋类）。
  - 避免长期大量食用卷心菜、木薯等致甲状腺肿食物（煮熟后影响小）。

---

### 三、TSH 降低的常见情况（提示甲状腺功能亢进倾向或垂体问题）

#### 1. 解读逻辑
TSH 降低意味着垂体"停止催促"，通常提示甲状腺激素过多（甲亢），或垂体本身功能衰竭。

#### 2. 可能原因
| 类型 | 具体原因 | 特点 |
|------|---------|------|
| **原发性甲亢**（最常见） | Graves 病（弥漫性毒性甲状腺肿） | 自身免疫病，伴 TSH 受体抗体（TRAb）阳性，可能有突眼、手抖 |
| | 毒性结节性甲状腺肿 | 甲状腺结节自主分泌激素 |
| | 甲状腺炎急性期（亚急性、无痛性、产后） | 甲状腺细胞破坏，激素漏出，呈"破坏性甲亢"，通常暂时性 |
| **亚临床甲亢** | TSH 降低（通常 0.1–0.4 或 <0.1），但 FT3/FT4 正常 | 早期甲亢或药物过量 |
| **药物性** | 过量服用甲状腺素（优甲乐）或含甲状腺激素的保健品 | 需调整剂量 |
| **继发性甲减**（垂体性） | 垂体功能减退、席汉综合征、垂体瘤术后 | TSH 低且 FT3/FT4 也低（与原发性甲亢相反），属于"中枢性甲减" |
| **非甲状腺疾病** | 严重全身性疾病、长期饥饿、使用糖皮质激素/多巴胺 | 低 T3 综合征的一部分，TSH 可轻度降低或正常低值 |

#### 3. 后续处理建议
- **鉴别诊断关键**：必须结合 **FT3、FT4** 判断：
  - 若 TSH 低 + FT3/FT4 **高** → 原发性甲亢（常见）
  - 若 TSH 低 + FT3/FT4 **低** → 垂体功能减退（危急，需内分泌科紧急处理）
- **进一步检查**：
  - **TRAb**（促甲状腺激素受体抗体）和 **TPOAb**：确诊 Graves 病。
  - 甲状腺摄碘率（鉴别 Graves 病 vs 甲状腺炎：前者摄碘率高，后者低）。
  - 甲状腺超声 + 血流检查（Graves 病呈"火海征"，炎症可能伴疼痛）。
  - 若怀疑垂体问题：查垂体 MRI 和其他垂体激素（皮质醇、性激素、生长激素）。
- **治疗与随访**：
  - **Graves 病**：抗甲状腺药物（甲巯咪唑/丙硫氧嘧啶）、放射性碘治疗或手术。
  - **甲状腺炎**：通常无需抗甲亢药物（因激素是漏出而非合成过多），以对症治疗（β受体阻滞剂控制心率）为主，定期监测可能转为甲减。
  - **药物过量**：调整优甲乐剂量。
  - **亚临床甲亢**：若 TSH < 0.1 或伴有心脏病、骨质疏松风险，需治疗。
- **生活方式**：
  - **严格忌碘**：禁食海带、紫菜、海鱼，使用无碘盐，避免含碘造影剂和药物（如胺碘酮）。
  - 避免咖啡、浓茶、酒精和剧烈运动（控制心率）。
  - 保证热量和蛋白质摄入（甲亢代谢快）。

---

### 四、重要提醒

1.  **TSH 是筛查首选，但非唯一**：必须结合 **FT3（游离三碘甲状腺原氨酸）** 和 **FT4（游离甲状腺素）** 才能区分"临床甲减/甲亢"与"亚临床状态"，以及判断病变部位（甲状腺 vs 垂体）。
2.  **检测干扰**：生物素（维生素 B7，常见于护发保健品）可能严重干扰 TSH 和甲状腺激素检测结果，检测前应停用 48–72 小时。
3.  **动态变化**：急性疾病、应激、妊娠早期（hCG 刺激甲状腺）可能导致 TSH 暂时性波动，需在身体状态稳定后复查。
4.  **孕妇特殊标准**：妊娠期 TSH 参考范围与普通人不同，通常孕早期 TSH 上限为 2.5 mIU/L，中晚期为 3.0 mIU/L，需专科管理。

**总结**：TSH 异常是甲状腺发出的"求救信号"，但无需恐慌。发现异常后，建议至**内分泌科**就诊，通过"激素全套（TSH+FT3+FT4）+ 抗体 + 超声"的组合检查明确病因，绝大多数情况可通过药物或观察有效管理。
