**白蛋白/球蛋白比值（Albumin/Globulin Ratio，A/G 比值）** 是肝功能检查中的重要衍生指标，反映肝脏合成功能、营养状况及免疫状态。以下是详细解析：

---

## 一、基本涵义

### 1. 定义与计算
- **公式**：A/G = 白蛋白(Albumin) ÷ 球蛋白(Globulin)
- **正常参考范围**：**1.2–2.4**（或 1.5–2.5，不同实验室标准略有差异）
- **计算逻辑**：总蛋白(TP) = 白蛋白 + 球蛋白，因此球蛋白 = TP − 白蛋白

### 2. 生理意义
- **白蛋白**：由肝脏合成，维持血浆胶体渗透压，反映**肝脏合成功能**和**营养状态**
- **球蛋白**：主要由免疫系统产生（γ-球蛋白），反映**免疫活性**和**慢性炎症状态**
- **比值意义**：衡量白蛋白与球蛋白的相对平衡，比单独看某一项更敏感

---

## 二、常见异常及可能原因

### 🔴 **A/G 比值降低（<1.2，甚至<1.0）—— 最常见且临床意义更大**

**机制**：白蛋白减少 和/或 球蛋白升高

| 异常方向 | 常见疾病/原因 | 机制说明 |
|---------|-------------|---------|
| **白蛋白降低为主** | **慢性肝病/肝硬化** | 肝细胞受损，合成白蛋白能力下降 |
| | **肾病综合征** | 肾小球滤过屏障破坏，白蛋白从尿中大量丢失 |
| | **营养不良** | 蛋白质摄入不足（长期素食、厌食症、吸收不良综合征） |
| | **慢性消耗性疾病** | 恶性肿瘤、结核、严重感染导致蛋白分解加速 |
| **球蛋白升高为主** | **多发性骨髓瘤** | 浆细胞恶性增殖，产生大量异常单克隆免疫球蛋白 |
| | **自身免疫病** | 类风湿关节炎、SLE、硬皮病等产生大量自身抗体 |
| | **慢性感染** | 结核、疟疾、亚急性细菌性心内膜炎等慢性抗原刺激 |
| | **肝硬化代偿期** | 肝脏纤维化导致球蛋白代偿性升高，白蛋白相对降低 |

### 🟡 **A/G 比值升高（>2.5）—— 相对少见**

通常由**球蛋白绝对降低**导致：
- **先天性低球蛋白血症**（免疫缺陷）
- **长期使用免疫抑制剂**（激素、化疗药物）
- **放射治疗后**免疫功能抑制
- **肾病早期**（选择性丢失小分子球蛋白，但此情况较少见）

---

## 三、建议后续处理

### 1. 初步评估（先看绝对值）
**不要只看比值，必须结合白蛋白和球蛋白的绝对值：**
- 若白蛋白正常、球蛋白升高 → 提示免疫增殖或慢性炎症
- 若白蛋白降低、球蛋白正常 → 提示肝合成功能障碍或营养问题
- 若两者都异常 → 提示严重肝病（如肝硬化）

### 2. 进一步检查（根据怀疑病因）

| 怀疑方向 | 建议检查项目 |
|---------|-------------|
| **肝脏疾病** | 肝功能全套（ALT、AST、GGT、ALP、胆红素）、凝血功能、肝炎病毒标志物（乙肝两对半、丙肝抗体）、肝脏超声或 FibroScan、必要时肝穿刺 |
| **肾脏疾病** | 尿常规（尿蛋白定性）、24小时尿蛋白定量、肾功能（肌酐、尿素氮）、肾脏超声 |
| **免疫增殖病** | **血清蛋白电泳**（看M蛋白峰）、免疫固定电泳、血清游离轻链、骨髓穿刺（怀疑骨髓瘤时） |
| **自身免疫病** | 自身抗体谱（ANA、抗ds-DNA、RF、抗CCP抗体）、血沉、C反应蛋白 |
| **营养评估** | 前白蛋白（更敏感的营养指标）、膳食调查、微量元素检测 |

### 3. 生活方式调整建议

**针对 A/G 降低（最常见情况）：**

- **饮食调整**：
  - **增加优质蛋白**：鱼肉、鸡胸肉、鸡蛋、牛奶、豆制品（肾功能正常者每日蛋白摄入 1.2–1.5g/kg）
  - **补充支链氨基酸**：肝硬化患者可适当增加亮氨酸、异亮氨酸摄入
  - **限盐**：如有低白蛋白倾向，适当限盐减轻水肿风险

- **肝脏保护**（如怀疑肝源性）：
  - **绝对戒酒**：酒精直接损伤肝细胞，抑制白蛋白合成
  - **避免肝毒性药物**：如对乙酰氨基酚、某些抗生素、中草药（何首乌等）
  - **控制体重**：减轻脂肪肝风险

- **免疫调节**（如球蛋白异常升高）：
  - 治疗基础感染病灶（如口腔、呼吸道慢性感染）
  - 避免过度劳累，规律作息

### 4. 复查策略
- **轻度异常**（如 1.0–1.2）：1–3 个月后复查，排除一过性因素（急性感染、脱水）
- **明显倒置**（<1.0）：**必须立即就诊**，完善上述检查，排除肝硬化或骨髓瘤
- **动态监测**：慢性肝病患者应每 3–6 个月监测 A/G 比值，比值持续下降提示肝功能恶化

---

## 四、特别提醒

1. **A/G 倒置（<1.0）是肝硬化的特征性表现之一**，特别是当白蛋白 <35g/L 且球蛋白 >35g/L 时，高度提示肝硬化失代偿期。

2. **多发性骨髓瘤**常表现为白蛋白正常或轻度降低，但球蛋白显著升高（可达 50–80g/L），A/G 比值可低至 0.3–0.5，需通过蛋白电泳鉴别。

3. **妊娠期**可能出现生理性白蛋白降低（血液稀释），A/G 比值轻度下降（约 1.0–1.2），通常产后恢复。

**总结**：A/G 比值是反映机体蛋白代谢平衡的敏感指标，**比值降低比升高更具临床意义**。发现异常时，应结合临床表现和其他实验室检查综合分析，必要时由消化内科或血液科医生进一步诊治。
